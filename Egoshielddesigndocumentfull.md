1. Project Overview & Vision
1.1 What is EgoShield?

EgoShield is a local-first, real-time manipulation detection system that acts as a cognitive firewall between users and persuasive web content. It is an annotation tool, not a blocker — it surfaces manipulation tactics as they happen across:

    Web pages (articles, landing pages, checkout flows)
    Email (inbox monitoring via local IMAP)
    Social media feeds (inline detection in Twitter/Facebook/LinkedIn streams)
    System notifications (OS-level notification analysis)

The system runs entirely on the user's machine: a local Python backend daemon processes content, applies rule-based and ML-based detectors, and uses a local LLM (via Ollama) as a final arbiter to generate human-readable explanations. A browser extension captures content and displays inline alerts.

Core Philosophy: Mirror, don't muzzle. EgoShield does not block content or make decisions for the user — it makes the invisible visible by annotating manipulation tactics with evidence and explanation.
1.2 The Problem Being Solved

Modern persuasion is engineered, invisible, and relentless:

    Dark UX patterns: Pre-ticked checkboxes, confirm-shaming, fake urgency timers, misdirection
    Linguistic manipulation: Manufactured scarcity, loss aversion framing, emotional hijacking
    Social proof exploitation: Fake testimonials, manipulated follower counts, pressure tactics
    Gaslighting patterns: DARVO tactics, reality distortion, relationship manipulation (especially in customer service contexts)

These tactics compound when combined, are hard for users to spot in real-time, and operate at scale across the web. Regulators have documented this but users lack tools to defend themselves in the moment.

EgoShield gives users a real-time cognitive immune system that:

    Detects tactics as they load
    Explains why they're manipulative
    Stores history for pattern recognition
    Never censors or blocks

1.3 Design Philosophy
Principle	Description
Non-blocking	Never block content. Always annotate, explain, and defer to user judgment.
Local-first	All processing, LLM inference, and data storage happen on-device. No cloud, no telemetry.
Evidence-driven	Every alert must cite a triggering phrase, DOM element, or behavioral signal.
Explainable	Human-readable explanations generated by local LLM, not just labels.
Privacy-preserving	No content leaves the machine. IMAP credentials stored encrypted locally.
Tunable	Per-domain trust levels, per-tactic suppression, sensitivity thresholds.
Transparent	Full audit log of what was detected, when, and why.
2. Goals, Non-Goals & Constraints
2.1 Goals (In Scope)

✅ Detection across content types:

    Web page text + DOM structure analysis
    Email subject lines + body text (IMAP, local only)
    Social media feed content (injected via content scripts)
    OS-level notification text

✅ Multi-strategy detection:

    Regex-based linguistic pattern matching
    DOM heuristics for dark UX patterns
    Behavioral signals (countdown timers, pre-checked boxes)
    ML-based classifiers for outrage/clickbait/propaganda
    LLM arbiter for final verdict + explanation generation

✅ Real-time annotation:

    Browser overlay with manipulation score + tactic breakdown
    Inline highlighting of triggering text/elements
    Configurable notification levels (silent/badge/popup)

✅ Analytics & reporting:

    Per-domain manipulation profiles
    Daily/weekly exposure reports
    Tactic frequency histograms
    Trust level management (trusted/neutral/suspicious/hostile domains)

✅ Extensibility:

    Pluggable detector modules (community detectors via Python imports)
    Custom regex pattern library (user-defined)
    Configurable severity weights

✅ Platform support:

    macOS, Linux, Windows (backend daemon)
    Chrome, Firefox, Edge (browser extensions via MV3)

2.2 Non-Goals (Explicitly Out of Scope)

❌ Content blocking or filtering — EgoShield is not an ad blocker or content filter
❌ Behavioral intervention — No "nudges," no automatic actions, no behavior modification
❌ Cloud sync or social features — No sharing, no leaderboards, no network effects
❌ VPN or network-level interception — EgoShield operates at the application layer only
❌ Real-time fact-checking — Detecting how something is said, not whether it's true
❌ Sentiment analysis of user-generated content — Focus is on institutional/commercial persuasion
❌ Legal advice or rights determination — Informational only, not a legal tool
2.3 Constraints

    Must run locally: All LLM inference via Ollama (mistral/llama3 models), no API calls to external services
    Must be fast: Detection pipeline < 200ms P95 for page analysis (excluding LLM arbiter)
    Must be privacy-safe: Email credentials encrypted at rest, no plaintext storage of sensitive content
    Must not break pages: Extension must not interfere with page functionality or layout
    Must handle MV3 constraints: Service worker architecture, no persistent background page, CSP-compliant
    Must support offline mode: All functionality available without internet connection (after initial setup)

    else:

# egoshield/detection/engine.py

from typing import List, Dict, Tuple
from concurrent.futures import ThreadPoolExecutor, as_completed
import logging

from .linguistic import (
    UrgencyDetector,
    ScarcityDetector,
    SocialProofDetector,
    FearDetector,
    LossAversionDetector,
    ReciprocityDetector,
    AuthorityDetector,
    GaslightingDetector,
    EmotionalLoadingDetector
)
from .ux import (
    DarkPatternDetector,
    CountdownDetector,
    PretickDetector,
    ConfirmShamingDetector,
    MisdirectionDetector
)
from .content import (
    OutrageBaitDetector,
    ClickbaitDetector,
    PropagandaDetector
)
from .models import TacticResult

logger = logging.getLogger(__name__)

class DetectionEngine:
    def __init__(self, config):
        self.config = config
        self.detectors = self._initialize_detectors()
        self.executor = ThreadPoolExecutor(max_workers=8)
    
    def _initialize_detectors(self):
        """Load all detector modules"""
        return [
            # Linguistic detectors
            UrgencyDetector(self.config),
            ScarcityDetector(self.config),
            SocialProofDetector(self.config),
            FearDetector(self.config),
            LossAversionDetector(self.config),
            ReciprocityDetector(self.config),
            AuthorityDetector(self.config),
            GaslightingDetector(self.config),
            EmotionalLoadingDetector(self.config),
            
            # UX detectors
            DarkPatternDetector(self.config),
            CountdownDetector(self.config),
            PretickDetector(self.config),
            ConfirmShamingDetector(self.config),
            MisdirectionDetector(self.config),
            
            # Content detectors
            OutrageBaitDetector(self.config),
            ClickbaitDetector(self.config),
            PropagandaDetector(self.config),
        ]
    
    def analyze(self, content: Dict[str, str], dom_signals: Dict, url: str) -> List[TacticResult]:
        """
        Run all detectors in parallel and collect results.
        
        Args:
            content: {text, html, headline, cta, footer, etc.}
            dom_signals: {countdowns, pretickedBoxes, formDefaults, etc.}
            url: Page URL for context
        
        Returns:
            List of TacticResult objects (one per detected tactic)
        """
        results = []
        futures = {}
        
        # Submit all detector jobs
        for detector in self.detectors:
            future = self.executor.submit(detector.detect, content, dom_signals, url)
            futures[future] = detector.__class__.__name__
        
        # Collect results as they complete
        for future in as_completed(futures):
            detector_name = futures[future]
            try:
                tactic_results = future.result(timeout=self.config.detector_timeout_seconds)
                if tactic_results:
                    results.extend(tactic_results)
            except Exception as e:
                logger.error(f"{detector_name} failed: {e}", exc_info=True)
                # Continue with other detectors - never fail the whole pipeline
        
        return results
    
    def calculate_score(self, tactics: List[TacticResult]) -> Tuple[float, str]:
        """
        Aggregate individual tactic scores into overall manipulation score.
        
        Scoring model:
        - Base score = weighted average of tactic confidences
        - Compounding bonus for tactic combinations
        - Severity threshold mapping
        """
        if not tactics:
            return 0.0, "none"
        
        # Weight by severity
        severity_weights = {"low": 0.5, "medium": 1.0, "high": 1.5}
        weighted_sum = sum(
            t.confidence * severity_weights.get(t.severity, 1.0)
            for t in tactics
        )
        avg_score = weighted_sum / len(tactics)
        
        # Compounding factor (multiple tactics = higher risk)
        compounding_factor = min(1 + (len(tactics) - 1) * 0.1, 1.5)
        final_score = min(avg_score * compounding_factor, 1.0)
        
        # Map to severity
        if final_score < 0.3:
            severity = "low"
        elif final_score < 0.6:
            severity = "medium"
        else:
            severity = "high"
        
        return final_score, severity

# egoshield/detection/linguistic/urgency.py

import re
from typing import List, Dict
from ..models import TacticResult

class UrgencyDetector:
    """Detects manufactured urgency through temporal pressure language."""
    
    # Regex patterns (case-insensitive)
    PATTERNS = [
        r'\bact\s+now\b',
        r'\blimited\s+time\b',
        r'\btoday\s+only\b',
        r'\bhurry\b',
        r'\bexpires?\s+(soon|today|tonight|in\s+\d+\s+(hours?|minutes?))\b',
        r'\blast\s+chance\b',
        r'\bdon\'t\s+miss\s+out\b',
        r'\bwhile\s+supplies?\s+last\b',
        r'\bending\s+soon\b',
        r'\bdeadline\s+(approaching|today|tonight)\b',
    ]
    
    def __init__(self, config):
        self.config = config
        self.compiled_patterns = [re.compile(p, re.IGNORECASE) for p in self.PATTERNS]
    
    def detect(self, content: Dict[str, str], dom_signals: Dict, url: str) -> List[TacticResult]:
        """
        Scan content for urgency patterns.
        
        Returns:
            List with 0 or 1 TacticResult (aggregated if multiple matches)
        """
        all_text = " ".join(content.values())
        matches = []
        
        for pattern in self.compiled_patterns:
            found = pattern.findall(all_text)
            matches.extend(found)
        
        if not matches:
            return []
        
        # Calculate confidence based on match count
        confidence = min(len(matches) * 0.2, 0.95)
        
        # Extract first match as evidence
        evidence = matches[0][:100]  # Truncate if very long
        
        # Determine location (headline vs body)
        location = "headline" if any(
            pattern.search(content.get("headline", ""))
            for pattern in self.compiled_patterns
        ) else "body"
        
        return [TacticResult(
            type="urgency",
            confidence=confidence,
            severity="medium",
            evidence=f'"{evidence}"' + (f" (and {len(matches)-1} more)" if len(matches) > 1 else ""),
            explanation=(
                "This content uses time-pressure language to create a sense of urgency. "
                "Phrases like 'act now' or 'limited time' are designed to make you feel "
                "you must decide immediately, potentially before thinking it through."
            ),
            location=location
        )]

from typing import List, Dict
from ..models import TacticResult

class DarkPatternDetector:
    """Detects UX dark patterns from DOM signals sent by extension."""
    
    def __init__(self, config):
        self.config = config
    
    def detect(self, content: Dict[str, str], dom_signals: Dict, url: str) -> List[TacticResult]:
        """
        Analyze DOM signals for dark UX patterns.
        
        dom_signals structure:
        {
            "pretickedBoxes": [
                {"id": "newsletter-optin", "label": "Send me updates", "checked": true}
            ],
            "countdowns": [
                {"element": "div.timer", "resetsOnReload": true, "initialValue": "10:00"}
            ],
            "confirmShaming": [
                {"buttonText": "No thanks, I hate saving money", "action": "reject"}
            ],
            "hiddenCosts": [
                {"element": "span.fee", "text": "Processing fee: $9.99", "hiddenUntil": "checkout"}
            ]
        }
        """
        results = []
        
        # Pre-ticked boxes
        if preticked := dom_signals.get("pretickedBoxes", []):
            results.append(TacticResult(
                type="preticked_defaults",
                confidence=0.95,
                severity="high",
                evidence=f"{len(preticked)} checkbox(es) pre-selected: " + 
                         ", ".join([f'"{p["label"]}"' for p in preticked[:2]]),
                explanation=(
                    "Checkboxes are pre-selected by default, likely enrolling you in "
                    "services or subscriptions without explicit opt-in. This exploits "
                    "inertia and makes it easy to accidentally consent."
                ),
                location="form"
            ))
        
        # Fake countdown timers
        if countdowns := dom_signals.get("countdowns", []):
            fake_countdowns = [c for c in countdowns if c.get("resetsOnReload")]
            if fake_countdowns:
                results.append(TacticResult(
                    type="fake_urgency_timer",
                    confidence=0.90,
                    severity="high",
                    evidence=f"Countdown timer resets on page reload (detected {len(fake_countdowns)} timer(s))",
                    explanation=(
                        "This countdown timer resets when you reload the page, meaning it's "
                        "not actually tracking a real deadline. It's a fake urgency tactic "
                        "designed to pressure you into a hasty decision."
                    ),
                    location="cta"
                ))
        
        # Confirm-shaming
        if shaming := dom_signals.get("confirmShaming", []):
            results.append(TacticResult(
                type="confirm_shaming",
                confidence=0.95,
                severity="medium",
                evidence=f'Reject button says: "{shaming[0]["buttonText"]}"',
                explanation=(
                    "The rejection option uses guilt-inducing language to make you feel "
                    "bad about declining. This emotional manipulation is designed to "
                    "pressure you into accepting against your better judgment."
                ),
                location="cta"
            ))
        
        # Hidden costs
        if hidden := dom_signals.get("hiddenCosts", []):
            results.append(TacticResult(
                type="hidden_costs",
                confidence=0.85,
                severity="high",
                evidence=f"{len(hidden)} cost(s) not visible until checkout",
                explanation=(
                    "Additional fees or costs are hidden until late in the purchase process. "
                    "This 'drip pricing' tactic exploits the sunk cost fallacy by revealing "
                    "extra charges after you've invested time in the process."
                ),
                location="checkout"
            ))
        
        return results

Arbiter Implementation

# egoshield/arbiter/llm_arbiter.py

import httpx
import json
from typing import List, Dict
from pydantic import BaseModel

class ArbiterResult(BaseModel):
    tactics: List[Dict]  # Re-ranked/filtered TacticResult dicts
    explanation: str
    confidence_adjustments: Dict[str, float]

class LLMArbiter:
    """
    Uses local LLM (via Ollama) to:
    1. Validate that detected tactics are contextually appropriate
    2. Generate user-friendly explanations
    3. Re-rank tactics by relevance
    4. Filter out false positives
    """
    
    SYSTEM_PROMPT = """You are an expert analyst of persuasive communication and dark patterns.

Your role is to validate detected manipulation tactics and explain them clearly.

Guidelines:
- Be objective and evidence-based
- Explain tactics in plain language
- Cite specific phrases or elements from the content
- Distinguish between legitimate marketing and manipulative tactics
- If a detected tactic seems like a false positive, note that
- Keep explanations under 150 words total
- Use a calm, informative tone (not alarmist)"""
    
    def __init__(self, base_url: str = "http://localhost:11434", model: str = "mistral:7b"):
        self.base_url = base_url
        self.model = model
        self.client = httpx.AsyncClient(timeout=30.0)
    
    async def refine(self, tactics: List, content_sample: str) -> ArbiterResult:
        """
        Send detected tactics + content sample to LLM for refinement.
        
        Args:
            tactics: List of TacticResult objects from detectors
            content_sample: First ~2000 chars of page content for context
        
        Returns:
            ArbiterResult with refined tactics and explanation
        """
        # Build prompt
        tactics_summary = "\n".join([
            f"- {t.type} (confidence: {t.confidence:.2f}, evidence: {t.evidence[:80]})"
            for t in tactics[:10]  # Limit to avoid context overflow
        ])
        
        user_prompt = f"""Content sample:
---
{content_sample}
---

Detected tactics:
{tactics_summary}

Task:
1. For each tactic, assess if it's a true positive or false positive given the content context
2. Provide a brief, user-friendly explanation (2-3 sentences) of what these tactics are trying to do
3. Return JSON with:
   - "validated_tactics": list of tactic types that are true positives
   - "explanation": combined explanation for the user
   - "adjustments": dict of confidence adjustments (e.g. {{"urgency": -0.2}} if overstated)

JSON only, no other text."""
        
        # Call Ollama API
        try:
            response = await self.client.post(
                f"{self.base_url}/api/generate",
                json={
                    "model": self.model,
                    "prompt": f"{self.SYSTEM_PROMPT}\n\n{user_prompt}",
                    "stream": False,
                    "format": "json"
                }
            )
            response.raise_for_status()
            result = response.json()
            llm_output = json.loads(result["response"])
            
            # Parse LLM output
            validated = set(llm_output.get("validated_tactics", []))
            adjustments = llm_output.get("adjustments", {})
            explanation = llm_output.get("explanation", "")
            
            # Filter and adjust tactics
            refined_tactics = []
            for tactic in tactics:
                if tactic.type in validated:
                    # Apply confidence adjustment if suggested
                    if tactic.type in adjustments:
                        tactic.confidence = max(0.0, min(1.0, tactic.confidence + adjustments[tactic.type]))
                    refined_tactics.append(tactic)
            
            return ArbiterResult(
                tactics=refined_tactics,
                explanation=explanation,
                confidence_adjustments=adjustments
            )
        
        except Exception as e:
            # Fallback: return original tactics if LLM call fails
            print(f"LLM arbiter error: {e}")
            return ArbiterResult(
                tactics=tactics,
                explanation="(LLM explanation unavailable)",
                confidence_adjustments={}
            )

interface DOMSignals {
  pretickedBoxes: Array<{id: string; label: string; checked: boolean}>;
  countdowns: Array<{element: string; resetsOnReload: boolean; initialValue: string}>;
  confirmShaming: Array<{buttonText: string; action: string}>;
  hiddenCosts: Array<{element: string; text: string; hiddenUntil: string}>;
}

interface AnalysisRequest {
  url: string;
  title: string;
  domain: string;
  source_type: string;
  content: {
    text: string;
    html: string;
    headline: string;
    cta: string;
    footer: string;
  };
  dom_signals: DOMSignals;
}

class PageAnalyzer {
  private backendUrl = 'http://localhost:9765';
  
  async analyzePage(): Promise<void> {
    // Extract page content
    const content = this.extractContent();
    const domSignals = this.detectDOMSignals();
    
    // Build request
    const request: AnalysisRequest = {
      url: window.location.href,
      title: document.title,
      domain: window.location.hostname,
      source_type: 'webpage',
      content,
      dom_signals: domSignals
    };
    
    // Send to backend
    try {
      const response = await fetch(`${this.backendUrl}/api/v1/analyze`, {
        method: 'POST',
        headers: {'Content-Type': 'application/json'},
        body: JSON.stringify(request)
      });
      
      if (!response.ok) {
        throw new Error(`Backend error: ${response.status}`);
      }
      
      const result = await response.json();
      
      // Display overlay if score > threshold
      if (result.score > 0.2) {
        this.showOverlay(result);
      }
      
      // Update badge
      chrome.runtime.sendMessage({
        type: 'UPDATE_BADGE',
        score: result.score,
        tacticCount: result.tactic_count
      });
      
    } catch (error) {
      console.error('EgoShield analysis failed:', error);
      // Fail silently - never break the page
    }
  }
  
  private extractContent(): AnalysisRequest['content'] {
    // Main text content
    const bodyText = document.body.innerText;
    
    // Headline (try common selectors)
    const headlineEl = document.querySelector('h1, .headline, .title, [class*="heading"]');
    const headline = headlineEl?.textContent || '';
    
    // CTA buttons
    const ctaButtons = Array.from(
      document.querySelectorAll('button, a[class*="cta"], [class*="buy"], [class*="checkout"]')
    );
    const cta = ctaButtons.map(b => b.textContent?.trim()).join(' | ');
    
    // Footer
    const footerEl = document.querySelector('footer, [role="contentinfo"]');
    const footer = footerEl?.textContent || '';
    
    return {
      text: bodyText,
      html: document.documentElement.outerHTML.slice(0, 50000), // Truncate
      headline,
      cta,
      footer
    };
  }
  
  private detectDOMSignals(): DOMSignals {
    return {
      pretickedBoxes: this.findPretickedBoxes(),
      countdowns: this.findCountdowns(),
      confirmShaming: this.findConfirmShaming(),
      hiddenCosts: this.findHiddenCosts()
    };
  }
  
  private findPretickedBoxes(): DOMSignals['pretickedBoxes'] {
    const checkboxes = Array.from(document.querySelectorAll('input[type="checkbox"]:checked'));
    return checkboxes.map(cb => {
      const label = document.querySelector(`label[for="${cb.id}"]`) || 
                    cb.closest('label') ||
                    cb.parentElement;
      return {
        id: cb.id || 'unlabeled',
        label: label?.textContent?.trim() || '',
        checked: true
      };
    });
  }
  
  private findCountdowns(): DOMSignals['countdowns'] {
    // Look for timer-like elements
    const timerElements = Array.from(
      document.querySelectorAll('[class*="timer"], [class*="countdown"], [id*="timer"]')
    );
    
    return timerElements.map(el => {
      const initialValue = el.textContent?.trim() || '';
      // Simple heuristic: if it's a static number, likely fake
      const resetsOnReload = /^\d{1,2}:\d{2}/.test(initialValue);
      
      return {
        element: this.getSelector(el),
        resetsOnReload,
        initialValue
      };
    });
  }
  
  private findConfirmShaming(): DOMSignals['confirmShaming'] {
    const shamingKeywords = ['no thanks', 'not interested', 'skip', 'decline', 'reject'];
    const buttons = Array.from(document.querySelectorAll('button, a[role="button"]'));
    
    return buttons
      .filter(btn => {
        const text = btn.textContent?.toLowerCase() || '';
        return shamingKeywords.some(kw => text.includes(kw)) && 
               (text.length > 20 || /\b(hate|stupid|fool)\b/i.test(text));
      })
      .map(btn => ({
        buttonText: btn.textContent?.trim() || '',
        action: 'reject'
      }));
  }
  
  private findHiddenCosts(): DOMSignals['hiddenCosts'] {
    // This is a simple heuristic - real implementation would be more sophisticated
    const costElements = Array.from(
      document.querySelectorAll('[class*="fee"], [class*="charge"], [class*="cost"]')
    );
    
    return costElements
      .filter(el => {
        const style = window.getComputedStyle(el);
        return style.display === 'none' || style.visibility === 'hidden';
      })
      .map(el => ({
        element: this.getSelector(el),
        text: el.textContent?.trim() || '',
        hiddenUntil: 'checkout'
      }));
  }
  
  private getSelector(element: Element): string {
    if (element.id) return `#${element.id}`;
    if (element.className) return `.${element.className.split(' ')[0]}`;
    return element.tagName.toLowerCase();
  }
  
  private showOverlay(result: any): void {
    // Create/update overlay iframe
    let overlay = document.getElementById('egoshield-overlay') as HTMLIFrameElement;
    if (!overlay) {
      overlay = document.createElement('iframe');
      overlay.id = 'egoshield-overlay';
      overlay.src = chrome.runtime.getURL('overlay.html');
      overlay.style.cssText = `
        position: fixed;
        top: 20px;
        right: 20px;
        width: 380px;
        height: auto;
        max-height: 600px;
        border: none;
        border-radius: 12px;
        box-shadow: 0 8px 32px rgba(0,0,0,0.3);
        z-index: 2147483647;
        background: white;
      `;
      document.body.appendChild(overlay);
    }
    
    // Send data to overlay iframe
    overlay.addEventListener('load', () => {
      overlay.contentWindow?.postMessage({
        type: 'ANALYSIS_RESULT',
        data: result
      }, '*');
    });
  }
}

Email Reader (IMAP)

# egoshield/capture/email_reader.py

import imaplib
import email
from email.header import decode_header
from typing import List, Dict
import logging
from threading import Thread
import time

logger = logging.getLogger(__name__)

class EmailReader:
    """
    Monitors IMAP inbox for new emails and analyzes them for manipulation.
    Runs in background thread, polls every N seconds.
    """
    
    def __init__(self, config, detector_engine, db):
        self.config = config
        self.detector = detector_engine
        self.db = db
        self.running = False
        self.thread = None
    
    def start(self):
        """Start background email monitoring thread"""
        if not self.config.email_monitoring_enabled:
            logger.info("Email monitoring disabled in config")
            return
        
        self.running = True
        self.thread = Thread(target=self._monitor_loop, daemon=True)
        self.thread.start()
        logger.info("Email monitoring started")
    
    def stop(self):
        """Stop monitoring thread"""
        self.running = False
        if self.thread:
            self.thread.join(timeout=5)
    
    def _monitor_loop(self):
        """Main monitoring loop (polls IMAP inbox)"""
        last_check = None
        
        while self.running:
            try:
                # Connect to IMAP server
                mail = imaplib.IMAP4_SSL(
                    self.config.email_imap_host,
                    self.config.email_imap_port
                )
                mail.login(
                    self.config.email_username,
                    self.config.email_password  # Encrypted in config
                )
                mail.select('INBOX')
                
                # Search for unseen emails since last check
                search_criteria = 'UNSEEN'
                if last_check:
                    search_criteria += f' SINCE {last_check.strftime("%d-%b-%Y")}'
                
                status, messages = mail.search(None, search_criteria)
                email_ids = messages[0].split()
                
                # Analyze each new email
                for email_id in email_ids[-10:]:  # Limit to 10 most recent
                    try:
                        status, msg_data = mail.fetch(email_id, '(RFC822)')
                        email_body = msg_data[0][1]
                        email_message = email.message_from_bytes(email_body)
                        
                        # Extract subject and body
                        subject = self._decode_header(email_message['Subject'])
                        sender = email_message['From']
                        body = self._get_email_body(email_message)
                        
                        # Analyze
                        result = self.detector.analyze(
                            content={
                                'text': body,
                                'headline': subject,
                                'html': '',
                                'cta': '',
                                'footer': ''
                            },
                            dom_signals={},
                            url=f"email://{sender}"
                        )
                        
                        # Store if manipulation detected
                        if result:
                            score, severity = self.detector.calculate_score(result)
                            if score > 0.3:
                                self.db.store_analysis(
                                    url=f"email://{sender}",
                                    domain=sender,
                                    source_type='email',
                                    score=score,
                                    severity=severity,
                                    tactic_count=len(result),
                                    tactics=result,
                                    processing_time_ms=0
                                )
                                logger.info(f"Manipulative email detected: {subject[:50]}")
                    
                    except Exception as e:
                        logger.error(f"Error analyzing email {email_id}: {e}")
                
                mail.close()
                mail.logout()
                last_check = time.time()
                
            except Exception as e:
                logger.error(f"IMAP connection error: {e}")
            
            # Sleep before next poll
            time.sleep(self.config.email_check_interval_seconds)
    
    def _decode_header(self, header_value):
        """Decode MIME header"""
        if not header_value:
            return ""
        decoded = decode_header(header_value)
        return "".join([
            text.decode(encoding or 'utf-8') if isinstance(text, bytes) else text
            for text, encoding in decoded
        ])
    
    def _get_email_body(self, email_message):
        """Extract plain text body from email"""
        body = ""
        if email_message.is_multipart():
            for part in email_message.walk():
                if part.get_content_type() == "text/plain":
                    body = part.get_payload(decode=True).decode('utf-8', errors='ignore')
                    break
        else:
            body = email_message.get_payload(decode=True).decode('utf-8', errors='ignore')
        return body

    def load(cls, config_path: str = None) -> 'Config':
        """Load config from TOML file, falling back to defaults"""
        if config_path is None:
            config_path = cls.default_config_path()
        
        config_file = Path(config_path).expanduser()
        
        if not config_file.exists():
            # Create default config
            config = cls()
            config.save(config_file)
            return config
        
        # Load from file
        data = toml.load(config_file)
        
        return cls(
            backend=BackendConfig(**data.get('backend', {})),
            detection=DetectionConfig(**data.get('detection', {})),
            llm=LLMConfig(**data.get('llm', {})),
            database=DatabaseConfig(**data.get('database', {})),
            capture=CaptureConfig(**data.get('capture', {})),
            dashboard=DashboardConfig(**data.get('dashboard', {})),
            extension=ExtensionConfig(**data.get('extension', {})),
            privacy=PrivacyConfig(**data.get('privacy', {})),
            rules=RulesConfig(**data.get('rules', {})),
        )
    
    def save(self, path: Path):
        """Write current config to file"""
        path.parent.mkdir(parents=True, exist_ok=True)
        with open(path, 'w') as f:
            toml.dump(self.to_dict(), f)
    
    def to_dict(self) -> Dict:
        """Convert to dict for TOML serialization"""
        return {
            'backend': self.backend.__dict__,
            'detection': self.detection.__dict__,
            'llm': self.llm.__dict__,
            'database': self.database.__dict__,
            'capture': self.capture.__dict__,
            'dashboard': self.dashboard.__dict__,
            'extension': self.extension.__dict__,
            'privacy': self.privacy.__dict__,
            'rules': self.rules.__dict__,
        }

        """Platform-specific config path"""
        if os.name == 'nt':  # Windows
            return os.path.join(os.environ['APPDATA'], 'egoshield', 'config.toml')
        else:  # macOS / Linux

import re

class UrgencyDetector:
    """
    Detects manufactured urgency through temporal pressure language.
    
    Strategy:
    - Pattern matching for time-pressure phrases
    - Context analysis (urgency in headlines = higher severity)
    - Compound detection (multiple urgency signals = higher confidence)
    """
    
    # Core patterns (case-insensitive)
    PATTERNS = {
        'explicit_time': [
            r'\bact\s+now\b',
            r'\bhurry\b',
            r'\btoday\s+only\b',
            r'\btonight\s+only\b',
            r'\blast\s+chance\b',
            r'\bfinal\s+(hours?|minutes?|days?)\b',
            r'\bexpires?\s+(soon|today|tonight|tomorrow)\b',
            r'\bending\s+(soon|today|tonight)\b',
            r'\btime\s+is\s+running\s+out\b',
            r'\bdon\'t\s+wait\b',
        ],
        'countdown': [
            r'\b(only\s+)?\d+\s+(hours?|minutes?|days?)\s+(left|remaining)\b',
            r'\bexpires?\s+in\s+\d+\s+(hours?|minutes?|days?)\b',
            r'\b\d+:\d{2}(:\d{2})?\s+(remaining|left)\b',  # Timer format
        ],
        'scarcity_urgency': [

            r'\bgoing\s+(fast|quickly)\b',
            r'\balmost\s+gone\b',

            r'\bnever\s+again\b',
            r'\bone[-\s]?time\s+(offer|opportunity|deal)\b',
        ]
    }
    
    # Intensity multipliers
    INTENSITY_MULTIPLIERS = {
        'multiple_caps': 1.2,      # URGENT! LAST CHANCE!
        'exclamation_marks': 1.15,  # Act now!!!
        'red_text': 1.1,            # Detected via DOM signals
        'headline_placement': 1.3   # In headline/hero area
    }

        self.config = config
        self.compiled_patterns = {
            category: [re.compile(p, re.IGNORECASE) for p in patterns]
            for category, patterns in self.PATTERNS.items()
        }
    
    def detect(self, content: Dict[str, str], dom_signals: Dict, url: str) -> List[TacticResult]:
        all_text = " ".join(content.values())
        headline = content.get('headline', '')
        cta = content.get('cta', '')
        
        # Collect all matches by category
        matches_by_category = {}
        total_matches = []
        
        for category, patterns in self.compiled_patterns.items():
            category_matches = []
            for pattern in patterns:
                found = pattern.findall(all_text)
                category_matches.extend(found)
            if category_matches:
                matches_by_category[category] = category_matches
                total_matches.extend(category_matches)
        
        if not total_matches:
            return []
        
        # Calculate base confidence
        # More matches = higher confidence, but with diminishing returns
        base_confidence = min(0.5 + (len(total_matches) * 0.15), 0.95)
        
        # Apply intensity multipliers
        intensity = 1.0
        
        # Check for ALL CAPS
        if re.search(r'[A-Z]{4,}', all_text):
            intensity *= self.INTENSITY_MULTIPLIERS['multiple_caps']
        
        # Check for excessive exclamation marks
        exclamation_count = all_text.count('!')
        if exclamation_count >= 3:
            intensity *= self.INTENSITY_MULTIPLIERS['exclamation_marks']
        
        # Check if urgency is in headline (most visible)
        in_headline = any(
            any(p.search(headline) for p in patterns)
            for patterns in self.compiled_patterns.values()
        )
        if in_headline:
            intensity *= self.INTENSITY_MULTIPLIERS['headline_placement']
        
        final_confidence = min(base_confidence * intensity, 0.98)
        
        # Determine severity
        if final_confidence > 0.7 or len(matches_by_category) >= 3:
            severity = 'high'
        elif final_confidence > 0.5:
            severity = 'medium'
        else:
            severity = 'low'
        
        # Build evidence string
        evidence_samples = total_matches[:3]  # Show up to 3 examples
        evidence = ', '.join(f'"{m}"' for m in evidence_samples)
        if len(total_matches) > 3:
            evidence += f' (+{len(total_matches) - 3} more)'
        
        # Location
        location = 'headline' if in_headline else ('cta' if any(
            any(p.search(cta) for p in patterns)
            for patterns in self.compiled_patterns.values()
        ) else 'body')
        
        # Generate contextual explanation
        explanation = self._generate_explanation(matches_by_category, in_headline)
        
        return [TacticResult(
            type='urgency',
            confidence=final_confidence,
            severity=severity,
            evidence=evidence,
            explanation=explanation,
            location=location
        )]
    
    def _generate_explanation(self, matches_by_category: Dict, in_headline: bool) -> str:
        """Generate context-aware explanation based on which patterns matched"""
        base = "This content uses time-pressure language to create artificial urgency."
        
        details = []
        if 'explicit_time' in matches_by_category:
            details.append("direct commands to 'act now'")
        if 'countdown' in matches_by_category:
            details.append("countdown timers")
        if 'deadline' in matches_by_category:
            details.append("arbitrary deadlines")
        if 'fomo' in matches_by_category:
            details.append("fear-of-missing-out framing")
        
        if details:
            base += f" It employs {', '.join(details)} "
        
        base += "to pressure you into making a hasty decision without time to evaluate. "
        
        if in_headline:
            base += "The urgency is prominently displayed, maximizing pressure."
        
        return base

import re
from typing import List, Dict

class OutrageBaitDetector:
    """
    Detects content designed to provoke outrage/anger for engagement.
    
    Strategy:
    - Emotional intensity keywords (loaded language)
    - Absolutist framing ("always", "never")
    - Us-vs-them binary framings
    - Strawman characterizations
    - Sensationalist language
    
    Note: This is a heuristic detector. A full ML model would be trained on
    labeled outrage-bait vs. neutral content.
    """
    
    EMOTIONAL_INTENSITY = [
        r'\boutrageous\b', r'\bshocking\b', r'\binsane\b', r'\bcrazy\b',
        r'\bscandal\b', r'\bdisgusting\b', r'\bappalling\b', r'\bhorrific\b',
        r'\binfuriating\b', r'\benraging\b', r'\bunbelievable\b',
        r'\bdevastating\b', r'\balarming\b', r'\bterrifying\b',
    ]
    
    ABSOLUTIST_LANGUAGE = [
        r'\ball\s+\w+\s+are\b',  # "all X are Y"
        r'\bevery\s+\w+\s+(is|are)\b',
        r'\bno\s+(one|body)\s+should\b',
        r'\balways\b', r'\bnever\b',
        r'\beverything\b', r'\bnothing\b',
    ]
    
    US_VS_THEM = [
        r'\bthey\s+want\s+(you|us)\s+to\b',
        r'\bthey\s+don\'t\s+want\s+(you|us)\s+to\s+know\b',
        r'\bthey\'re\s+(trying|coming)\s+(to|for)\b',
        r'\bus\s+vs\.?\s+them\b',
        r'\bour\s+enemies\b',
    ]
    
    SENSATIONALISM = [
        r'\byou\s+won\'t\s+believe\b',
        r'\bwhat\s+happened\s+next\b',
        r'\bthis\s+changes\s+everything\b',
        r'\bthe\s+truth\s+they\s+don\'t\s+want\s+you\s+to\s+see\b',
        r'\bexposed\b', r'\brevealed\b', r'\buncovered\b',
    ]
    
    def __init__(self, config):
        self.config = config
        self.patterns = {
            'emotional': [re.compile(p, re.IGNORECASE) for p in self.EMOTIONAL_INTENSITY],
            'absolutist': [re.compile(p, re.IGNORECASE) for p in self.ABSOLUTIST_LANGUAGE],
            'us_vs_them': [re.compile(p, re.IGNORECASE) for p in self.US_VS_THEM],
            'sensational': [re.compile(p, re.IGNORECASE) for p in self.SENSATIONALISM],
        }
    
    def detect(self, content: Dict[str, str], dom_signals: Dict, url: str) -> List[TacticResult]:
        text = content.get('text', '') + ' ' + content.get('headline', '')
        
        # Count matches per category
        category_counts = {}
        all_matches = []
        
        for category, patterns in self.patterns.items():
            matches = []
            for pattern in patterns:
                found = pattern.findall(text)
                matches.extend(found)
            if matches:
                category_counts[category] = len(matches)
                all_matches.extend(matches[:2])  # Sample
        
        # Need hits in at least 2 categories to flag
        if len(category_counts) < 2:
            return []
        
        # Calculate confidence
        total_signals = sum(category_counts.values())
        confidence = min(0.5 + (total_signals * 0.08), 0.88)
        
        # Check for exclamation mark overuse
        exclamation_density = text.count('!') / max(len(text), 1) * 1000
        if exclamation_density > 5:  # More than 5 per 1000 chars
            confidence = min(confidence * 1.15, 0.92)
        
        severity = 'medium' if confidence < 0.7 else 'high'
        
        evidence = ', '.join(f'"{m}"' for m in all_matches[:3])
        
        explanation = (
            "This content uses inflammatory language and framing designed to provoke outrage "
            "rather than inform. "
        )
        
        if 'emotional' in category_counts:
            explanation += "Emotionally charged words amplify reactions. "
        if 'absolutist' in category_counts:
            explanation += "Absolute statements ('all', 'never') oversimplify complex issues. "
        if 'us_vs_them' in category_counts:
            explanation += "Us-vs-them framing encourages tribal thinking. "
        if 'sensational' in category_counts:
            explanation += "Sensationalist phrasing prioritizes clicks over accuracy. "
        
        explanation += "Outrage-driven content is often designed for engagement metrics, not nuanced understanding."
        
        return [TacticResult(
            type='outrage_bait',
            confidence=confidence,
            severity=severity,
            evidence=evidence,
            explanation=explanation.strip(),
            location='headline'
        )]

# egoshield/arbiter/llm_arbiter.py

import httpx
import json
import logging
from typing import List, Dict, Optional
from pydantic import BaseModel
from .prompts import ARBITER_SYSTEM_PROMPT, VALIDATION_PROMPT_TEMPLATE, EXPLANATION_PROMPT_TEMPLATE

logger = logging.getLogger(__name__)

class ArbiterResult(BaseModel):
    validated_tactics: List[str]
    false_positives: List[str]
    explanation: str
    confidence_adjustments: Dict[str, float]
    compounding_noted: bool

class LLMArbiter:
    """
    Local LLM arbiter via Ollama.
    
    Responsibilities:
    1. Validate rule-based detections (reduce false positives)
    2. Generate natural language explanations
    3. Detect tactic combinations/compounding effects
    4. Adjust confidence scores based on context
    """
    
    def __init__(self, base_url: str, model: str, timeout: int = 30):
        self.base_url = base_url
        self.model = model
        self.timeout = timeout
        self.client = httpx.AsyncClient(timeout=timeout)
        self._healthy = None
    
    async def health_check(self) -> bool:
        """Check if Ollama is reachable and model is available"""
        try:
            response = await self.client.get(f"{self.base_url}/api/tags")
            if response.status_code == 200:
                models = response.json().get('models', [])
                self._healthy = any(self.model in m['name'] for m in models)
                return self._healthy
        except Exception as e:
            logger.warning(f"Ollama health check failed: {e}")
            self._healthy = False
        return False
    
    async def refine(self, tactics: List[Dict], content_sample: str, headline: str = "") -> Optional[ArbiterResult]:
        """
        Validate and refine detected tactics.
        
        Returns None if LLM call fails (fallback to original tactics).
        """
        if not tactics:
            return None
        
        # Build tactics summary
        tactics_summary = "\n".join([
            f"- {t['type']} (confidence: {t['confidence']:.2f}): {t['evidence'][:80]}"
            for t in tactics
        ])
        
        # Build prompt
        prompt = VALIDATION_PROMPT_TEMPLATE.format(
            content_sample=content_sample,
            tactics_summary=tactics_summary
        )
        
        try:
            # Call Ollama
            response = await self.client.post(
                f"{self.base_url}/api/generate",
                json={
                    "model": self.model,
                    "system": ARBITER_SYSTEM_PROMPT,
                    "prompt": prompt,
                    "stream": False,
                    "format": "json",
                    "options": {
                        "temperature": 0.3,  # Lower temperature for more consistent JSON
                        "top_p": 0.9,
                    }
                }
            )
            response.raise_for_status()
            
            # Parse response
            result_json = response.json()
            llm_output = json.loads(result_json["response"])
            
            # Validate structure
            return ArbiterResult(
                validated_tactics=llm_output.get("validated_tactics", []),
                false_positives=llm_output.get("false_positives", []),
                explanation=llm_output.get("explanation", ""),
                confidence_adjustments=llm_output.get("confidence_adjustments", {}),
                compounding_noted=llm_output.get("compounding_noted", False)
            )
        
        except httpx.TimeoutException:
            logger.warning(f"LLM arbiter timeout after {self.timeout}s")
            return None
        except json.JSONDecodeError as e:
            logger.error(f"LLM returned invalid JSON: {e}")
            return None
        except Exception as e:
            logger.error(f"LLM arbiter error: {e}", exc_info=True)
            return None
    
    async def explain_tactics(self, tactics: List[Dict], headline: str) -> str:
        """
        Generate a plain-English explanation of detected tactics.
        Separate from validation - this is pure explanation generation.
        """
        tactics_list = "\n".join([
            f"- {t['type']}: {t['evidence'][:60]}"
            for t in tactics
        ])
        
        prompt = EXPLANATION_PROMPT_TEMPLATE.format(
            headline=headline,
            tactics_list=tactics_list
        )
        
        try:
            response = await self.client.post(
                f"{self.base_url}/api/generate",
                json={
                    "model": self.model,
                    "system": ARBITER_SYSTEM_PROMPT,
                    "prompt": prompt,
                    "stream": False,
                    "options": {"temperature": 0.5}
                }
            )
            response.raise_for_status()
            
            result_json = response.json()
            explanation = result_json["response"].strip()
            
            # Enforce length limit
            if len(explanation) > 500:
                explanation = explanation[:497] + "..."
            
            return explanation
        
        except Exception as e:
            logger.error(f"Explanation generation failed: {e}")
            return self._fallback_explanation(tactics)
    
    def _fallback_explanation(self, tactics: List[Dict]) -> str:
        """Generate explanation without LLM (fallback)"""
        if not tactics:
            return "No manipulation tactics detected."
        
        tactic_types = [t['type'].replace('_', ' ') for t in tactics]
        
        if len(tactics) == 1:
            return f"This content uses {tactic_types[0]} to influence your decision."
        elif len(tactics) == 2:
            return f"This content combines {tactic_types[0]} and {tactic_types[1]} to pressure you into action."
        else:
            return f"This content uses multiple manipulation tactics ({len(tactics)} detected) including {tactic_types[0]}, {tactic_types[1]}, and others to influence your behavior."
    
    async def close(self):
        """Cleanup"""
        await self.client.aclose()

import tiktoken
from typing import List, Dict

class ContextManager:
    """
    Manage LLM context windows to avoid exceeding token limits.
    
    Strategy:
    - Mistral 7B: ~8k token context (conservatively use 6k)
    - Llama 3 8B: ~8k token context (conservatively use 6k)
    - For longer content: chunk and summarize, or truncate intelligently
    """
    
    def __init__(self, model: str, max_tokens: int = 6000):
        self.model = model
        self.max_tokens = max_tokens
        
        # Use tiktoken for token estimation (gpt-4 encoding as proxy)
        try:
            self.encoding = tiktoken.get_encoding("cl100k_base")
        except:
            self.encoding = None
    
    def estimate_tokens(self, text: str) -> int:
        """Estimate token count"""
        if self.encoding:
            return len(self.encoding.encode(text))
        else:
            # Fallback: rough approximation (1 token ≈ 4 chars)
            return len(text) // 4
    
    def truncate_content(self, content: str, max_tokens: int) -> str:
        """Intelligently truncate content to fit token budget"""
        if self.estimate_tokens(content) <= max_tokens:
            return content
        
        # Strategy: Keep beginning and end, omit middle
        # Beginning often has headlines/key info, end has CTAs
        target_chars = max_tokens * 4  # Rough conversion
        
        keep_start = target_chars // 2
        keep_end = target_chars // 4
        
        if len(content) <= target_chars:
            return content
        
        truncated = (
            content[:keep_start] +
            "\n\n[... middle content omitted ...]\n\n" +
            content[-keep_end:]
        )
        
        return truncated
    
    def build_prompt(self, system: str, user: str, tactics: List[Dict]) -> Dict[str, int]:
        """
        Build prompt and return token budget breakdown.
        
        Returns:
            {
                'system_tokens': int,
                'user_tokens': int,
                'tactics_tokens': int,
                'total_tokens': int,
                'available_for_content': int
            }
        """
        system_tokens = self.estimate_tokens(system)
        
        # Tactics summary (fixed overhead)
        tactics_summary = "\n".join([
            f"- {t['type']}: {t['evidence'][:80]}"
            for t in tactics
        ])
        tactics_tokens = self.estimate_tokens(tactics_summary)
        
        # Reserve tokens for response (estimate ~300 tokens for JSON response)
        response_reserve = 300
        
        # Fixed parts of user prompt (template overhead)
        template_overhead = 150
        
        total_fixed = system_tokens + tactics_tokens + template_overhead + response_reserve
        available_for_content = max(self.max_tokens - total_fixed, 500)
        
        return {
            'system_tokens': system_tokens,
            'tactics_tokens': tactics_tokens,
            'template_overhead': template_overhead,
            'response_reserve': response_reserve,
            'total_fixed': total_fixed,
            'available_for_content': available_for_content,
            'max_tokens': self.max_tokens
        }

# egoshield/arbiter/fallback.py

from typing import List, Dict, Optional
from enum import Enum

class LLMStatus(Enum):
    HEALTHY = "healthy"
    DEGRADED = "degraded"
    UNAVAILABLE = "unavailable"

class FallbackStrategy:
    """
    Graceful degradation when LLM is unavailable or slow.
    
    Levels:
    1. Full LLM validation + explanation (ideal)
    2. Explanation only (validation skipped)
    3. Template-based explanation (no LLM)
    4. No explanation (detections only)
    """
    
    def __init__(self, arbiter):
        self.arbiter = arbiter
        self.status = LLMStatus.HEALTHY
        self.failure_count = 0
        self.success_count = 0
    
    async def execute_with_fallback(
        self,
        tactics: List[Dict],
        content_sample: str,
        headline: str,
        config: Dict
    ) -> Dict:
        """
        Execute LLM arbiter with automatic fallback on failure.
        
        Returns:
            {
                'tactics': List[Dict],  # Possibly filtered/adjusted
                'explanation': str,
                'llm_used': bool,
                'fallback_level': int  # 0 = no fallback, 1-3 = fallback levels
            }
        """
        # Check if LLM is disabled in config
        if not config.get('llm_enabled', True):
            return self._fallback_level_3(tactics, headline)
        
        # Level 0: Try full LLM validation + explanation
        if self.status == LLMStatus.HEALTHY or self.status == LLMStatus.DEGRADED:
            try:
                result = await self.arbiter.refine(tactics, content_sample, headline)
                
                if result:
                    # Success!
                    self.failure_count = 0
                    self.success_count += 1
                    self.status = LLMStatus.HEALTHY
                    
                    # Filter tactics based on LLM validation
                    validated = self._apply_validation(tactics, result)
                    
                    return {
                        'tactics': validated,
                        'explanation': result.explanation,
                        'llm_used': True,
                        'fallback_level': 0
                    }
            
            except Exception as e:
                self.failure_count += 1
                self.success_count = 0
                
                # Update status based on failure pattern
                if self.failure_count >= 3:
                    self.status = LLMStatus.UNAVAILABLE
                elif self.failure_count >= 1:
                    self.status = LLMStatus.DEGRADED
        
        # Level 1: Try explanation only (skip validation)
        if self.status == LLMStatus.DEGRADED:
            try:
                explanation = await self.arbiter.explain_tactics(tactics, headline)
                return {
                    'tactics': tactics,  # Use original detections
                    'explanation': explanation,
                    'llm_used': True,
                    'fallback_level': 1
                }
            except:
                pass
        
        # Level 2: Template-based explanation
        return self._fallback_level_3(tactics, headline)
    
    def _fallback_level_3(self, tactics: List[Dict], headline: str) -> Dict:
        """Level 3: No LLM, template-based explanation"""
        explanation = self._generate_template_explanation(tactics)
        
        return {
            'tactics': tactics,
            'explanation': explanation,
            'llm_used': False,
            'fallback_level': 3
        }
    
    def _generate_template_explanation(self, tactics: List[Dict]) -> str:
        """Generate explanation from templates (no LLM)"""
        if not tactics:
            return ""
        
        tactic_names = [t['type'].replace('_', ' ') for t in tactics]
        
        if len(tactics) == 1:
            return f"This content uses {tactic_names[0]}. {self._get_tactic_description(tactics[0]['type'])}"
        else:
            return (
                f"This content uses {len(tactics)} manipulation tactics including "
                f"{', '.join(tactic_names[:2])}. These work together to influence your behavior."
            )
    
    def _get_tactic_description(self, tactic_type: str) -> str:
        """One-sentence description per tactic type"""
        descriptions = {
            'urgency': "Time pressure language creates false urgency to rush your decision.",
            'scarcity': "Limited availability claims pressure you to act before thinking.",
            'social_proof': "References to others' behavior exploit the bandwagon effect.",
            'fear': "Fear-based framing triggers emotional rather than rational decisions.",
            'loss_aversion': "Framing emphasizes what you'll lose rather than gain.",
            'preticked_defaults': "Pre-selected options exploit user inertia.",
            'fake_countdown': "Countdown timers create fake urgency.",
            'confirm_shaming': "Guilt-inducing rejection options pressure acceptance.",
            'hidden_costs': "Costs hidden until late exploit the sunk cost fallacy.",
            'gaslighting': "Language patterns undermine your perception or memory.",
            'outrage_bait': "Inflammatory framing designed to provoke engagement over understanding.",
        }
        return descriptions.get(tactic_type, "This tactic is designed to influence your behavior.")
    
    def _apply_validation(self, tactics: List[Dict], validation_result) -> List[Dict]:
        """Filter and adjust tactics based on LLM validation"""
        validated_types = set(validation_result.validated_tactics)
        false_positives = set(validation_result.false_positives)
        
        filtered = []
        for tactic in tactics:
            if tactic['type'] in false_positives:
                continue  # Remove false positive
            
            if tactic['type'] in validated_types:
                # Apply confidence adjustment if present
                adjustment = validation_result.confidence_adjustments.get(tactic['type'], 0.0)
                tactic['confidence'] = max(0.0, min(1.0, tactic['confidence'] + adjustment))
                
                # Use LLM explanation if better than default
                if validation_result.explanation:
                    tactic['llm_explanation'] = validation_result.explanation
                
                filtered.append(tactic)
        
        return filtered

 * Runs in isolated context with DOM access but separate from page JS.
 * 
 * CSP Compliance:
 * - Cannot use eval() or inline scripts
 * - Cannot use external CDNs for libraries
 * - Must use chrome.runtime.getURL() for resources
 * - Cannot inject <script> tags into page
 */

import { AnalysisRequest, AnalysisResult } from './types';

class EgoShieldContentScript {
  private settings: any;
  private overlayIframe: HTMLIFrameElement | null = null;
  private analyzing: boolean = false;
  
  constructor() {
    this.init();
  }
  
  private async init(): Promise<void> {
    // Get settings from background
    this.settings = await this.getSettings();
    
    if (!this.settings.enabled) {
      return; // Extension disabled
    }
    
    // Wait for page to fully load
    if (document.readyState === 'loading') {
      document.addEventListener('DOMContentLoaded', () => this.analyzePage());
    } else {
      this.analyzePage();
    }
    
    // Listen for manual analysis trigger
    chrome.runtime.onMessage.addListener((message) => {
      if (message.type === 'ANALYZE_NOW') {
        this.analyzePage(true);
      }
    });
  }
  
  private async getSettings(): Promise<any> {
    return new Promise((resolve) => {
      chrome.runtime.sendMessage({ type: 'GET_SETTINGS' }, (settings) => {
        resolve(settings);
      });
    });
  }
  
  private async analyzePage(force: boolean = false): Promise<void> {
    if (this.analyzing) return;
    this.analyzing = true;
    
    try {
      const domain = window.location.hostname;
      
      // Check if domain is trusted (skip analysis)
      if (this.settings.trustedDomains?.includes(domain) && !force) {
        console.log('[EgoShield] Domain is trusted, skipping analysis');
        this.analyzing = false;
        return;
      }
      
      // Extract page content
      const content = this.extractContent();
      const domSignals = this.detectDOMSignals();
      
      // Build request
      const request: AnalysisRequest = {
        url: window.location.href,
        title: document.title,
        domain: domain,
        source_type: 'webpage',
        content,
        dom_signals: domSignals
      };
      
      // Send to backend (with CSP-safe fetch)
      const result = await this.sendToBackend(request);
      
      if (result && result.score > this.settings.scoreThreshold) {
        this.displayOverlay(result);
      }
      
      // Update badge via background script
      chrome.runtime.sendMessage({
        type: 'UPDATE_BADGE',
        data: {
          score: result?.score || 0,
          tacticCount: result?.tactic_count || 0
        }
      });
      
    } catch (error) {
      console.error('[EgoShield] Analysis failed:', error);
    } finally {
      this.analyzing = false;
    }
  }
  
  private extractContent(): AnalysisRequest['content'] {
    // Extract meaningful text while avoiding noise
    const bodyText = document.body.innerText;
    
    // Headline extraction (try multiple strategies)
    const headline = this.extractHeadline();
    
    // CTA extraction
    const cta = this.extractCTAs();
    
    // Footer
    const footerEl = document.querySelector('footer, [role="contentinfo"]');
    const footer = footerEl?.textContent?.trim() || '';
    
    // HTML (truncated for performance)
    const html = document.documentElement.outerHTML.slice(0, 50000);
    
    return {
      text: bodyText,
      html,
      headline,
      cta,
      footer
    };
  }
  
  private extractHeadline(): string {
    // Try multiple selectors in priority order
    const selectors = [
      'h1',
      '[class*="headline"]',
      '[class*="title"]',
      '[class*="heading"]',
      'header h2',
      'article h1',
      'main h1'
    ];
    
    for (const selector of selectors) {
      const el = document.querySelector(selector);
      if (el?.textContent) {
        return el.textContent.trim();
      }
    }
    
    // Fallback: title tag
    return document.title;
  }
  
  private extractCTAs(): string {
    const ctaSelectors = [
      'button',
      'a[href*="buy"]',
      'a[href*="signup"]',
      'a[href*="subscribe"]',
      '[class*="cta"]',
      '[class*="button"]',
      '[role="button"]'
    ];
    
    const ctas: string[] = [];
    for (const selector of ctaSelectors) {
      const elements = document.querySelectorAll(selector);
      elements.forEach((el) => {
        const text = el.textContent?.trim();
        if (text && text.length < 100) {
          ctas.push(text);
        }
      });
    }
    
    return ctas.slice(0, 10).join(' | ');
  }
  
  private detectDOMSignals(): AnalysisRequest['dom_signals'] {
    return {
      pretickedBoxes: this.findPretickedBoxes(),
      countdowns: this.findCountdowns(),
      confirmShaming: this.findConfirmShaming(),
      hiddenCosts: this.findHiddenCosts(),
      asymmetricButtons: this.findAsymmetricButtons(),
      fakeActivity: this.findFakeActivity()
    };
  }
  
  private findPretickedBoxes(): any[] {
    const checkboxes = Array.from(document.querySelectorAll<HTMLInputElement>('input[type="checkbox"]:checked'));
    
    return checkboxes.map(cb => {
      // Find associated label
      const label = document.querySelector(`label[for="${cb.id}"]`) ||
                    cb.closest('label') ||
                    cb.parentElement;
      
      return {
        id: cb.id || 'unknown',
        label: label?.textContent?.trim() || '',
        checked: true,
        name: cb.name || ''
      };
    });
  }
  
  private findCountdowns(): any[] {
    const timerElements = Array.from(
      document.querySelectorAll('[class*="timer"], [class*="countdown"], [id*="countdown"]')
    );
    
    return timerElements.map(el => {
      const text = el.textContent?.trim() || '';
      
      // Heuristic: if it matches HH:MM or MM:SS format, likely a timer
      const isTimerFormat = /\d{1,2}:\d{2}(:\d{2})?/.test(text);
      
      // Store initial value and check later if it resets
      const initialValue = text;
      
      // Check if it's an image (static, not real countdown)
      const isImage = el.tagName === 'IMG' || el.querySelector('img') !== null;
      
      return {
        element: this.getSelector(el),
        initialValue,
        isTimerFormat,
        isImage,
        resetsOnReload: false // Will be detected by backend tracking across sessions
      };
    });
  }
  
  private findConfirmShaming(): any[] {
    const shamingKeywords = ['no thanks', 'not interested', 'maybe later', 'decline', 'reject', 'skip'];
    const buttons = Array.from(document.querySelectorAll('button, a[role="button"], [class*="button"]'));
    
    return buttons
      .filter(btn => {
        const text = btn.textContent?.toLowerCase() || '';
        // Must contain a shaming keyword AND be longer than typical (suggests extra guilt-inducing text)
        return shamingKeywords.some(kw => text.includes(kw)) && text.length > 20;
      })
      .map(btn => ({
        buttonText: btn.textContent?.trim() || '',
        action: 'reject',
        selector: this.getSelector(btn)
      }));
  }
  
  private findHiddenCosts(): any[] {
    const costKeywords = ['fee', 'charge', 'cost', 'price', 'shipping', 'handling', 'tax'];
    const elements = Array.from(document.querySelectorAll('*')).filter(el => {
      const text = el.textContent?.toLowerCase() || '';
      return costKeywords.some(kw => text.includes(kw)) && text.length < 200;
    });
    
    return elements
      .filter(el => {
        const style = window.getComputedStyle(el);
        return style.display === 'none' || style.visibility === 'hidden' || parseFloat(style.opacity) < 0.1;
      })
      .map(el => ({
        element: this.getSelector(el),
        text: el.textContent?.trim() || '',
        hiddenUntil: 'checkout' // Heuristic
      }));
  }
  
  private findAsymmetricButtons(): any[] {
    // Look for modal dialogs with accept/reject buttons
    const modals = Array.from(document.querySelectorAll('[role="dialog"], [class*="modal"]'));
    
    const asymmetric: any[] = [];
    
    modals.forEach(modal => {
      const buttons = Array.from(modal.querySelectorAll('button, [role="button"]'));
      
      if (buttons.length >= 2) {
        // Compare button styling
        const styles = buttons.map(btn => {
          const computed = window.getComputedStyle(btn);
          return {
            element: btn,
            width: parseFloat(computed.width),
            height: parseFloat(computed.height),
            fontSize: parseFloat(computed.fontSize),
            color: computed.color,
            backgroundColor: computed.backgroundColor
          };
        });
        
        // Find accept vs reject buttons (heuristic based on text)
        const acceptBtn = buttons.find(b => /accept|agree|ok|yes|allow|enable/i.test(b.textContent || ''));
        const rejectBtn = buttons.find(b => /reject|decline|no|deny|disable/i.test(b.textContent || ''));
        
        if (acceptBtn && rejectBtn) {
          const acceptStyle = window.getComputedStyle(acceptBtn);
          const rejectStyle = window.getComputedStyle(rejectBtn);
          
          const acceptSize = parseFloat(acceptStyle.width) * parseFloat(acceptStyle.height);
          const rejectSize = parseFloat(rejectStyle.width) * parseFloat(rejectStyle.height);
          
          // If accept button is > 2x larger, flag it
          if (acceptSize > rejectSize * 2) {
            asymmetric.push({
              acceptSize: Math.round(acceptSize),
              rejectSize: Math.round(rejectSize),
              ratio: (acceptSize / rejectSize).toFixed(1),
              acceptText: acceptBtn.textContent?.trim(),
              rejectText: rejectBtn.textContent?.trim()
            });
          }
        }
      }
    });
    
    return asymmetric;
  }
  
  private findFakeActivity(): any[] {
    // Look for notification popups claiming "X just purchased"
    const activityKeywords = ['just purchased', 'just bought', 'just signed up', 'viewing this', 'people watching'];
    
    const notifications = Array.from(document.querySelectorAll('[class*="notification"], [class*="popup"], [class*="toast"]'));
    
    return notifications
      .filter(el => {
        const text = el.textContent?.toLowerCase() || '';
        return activityKeywords.some(kw => text.includes(kw));
      })
      .map(el => ({
        text: el.textContent?.trim() || '',
        selector: this.getSelector(el),
        timestamp: new Date().toISOString()
      }));
  }
  
  private getSelector(el: Element): string {
    if (el.id) return `#${el.id}`;
    if (el.className && typeof el.className === 'string') {
      const classes = el.className.split(' ').filter(c => c.length > 0);
      if (classes.length > 0) return `.${classes[0]}`;
    }
    return el.tagName.toLowerCase();
  }
  
  private async sendToBackend(request: AnalysisRequest): Promise<AnalysisResult | null> {
    try {
      const response = await fetch(`${this.settings.backendUrl}/api/v1/analyze`, {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
        },
        body: JSON.stringify(request)
      });
      
      if (!response.ok) {
        throw new Error(`Backend returned ${response.status}`);
      }
      
      return await response.json();
    } catch (error) {
      console.error('[EgoShield] Backend communication failed:', error);
      return null;
    }
  }
  
  private displayOverlay(result: AnalysisResult): void {
    // Remove existing overlay if present
    if (this.overlayIframe) {
      this.overlayIframe.remove();
    }
    
    // Create iframe for overlay (CSP-safe)
    this.overlayIframe = document.createElement('iframe');
    this.overlayIframe.id = 'egoshield-overlay';
    this.overlayIframe.src = chrome.runtime.getURL('overlay.html');
    
    // Styling based on settings
    const position = this.settings.overlayPosition || 'top-right';
    const positioning = this.getPositionCSS(position);
    
    this.overlayIframe.style.cssText = `
      position: fixed;
      ${positioning}
      width: 380px;
      max-height: 600px;
      border: none;
      border-radius: 12px;
      box-shadow: 0 8px 32px rgba(0,0,0,0.3);
      z-index: 2147483647;
      background: white;
      transition: transform 0.3s ease;
    `;
    
    document.body.appendChild(this.overlayIframe);
    
    // Send result to iframe once loaded
    this.overlayIframe.addEventListener('load', () => {
      this.overlayIframe?.contentWindow?.postMessage({
        type: 'ANALYSIS_RESULT',
        data: result
      }, '*');
    });
    
    // Listen for close message from overlay
    window.addEventListener('message', (event) => {
      if (event.data.type === 'CLOSE_OVERLAY') {
        this.overlayIframe?.remove();
        this.overlayIframe = null;
      }
    });
  }
  
  private getPositionCSS(position: string): string {
    switch (position) {
      case 'top-right':
        return 'top: 20px; right: 20px;';
      case 'top-left':
        return 'top: 20px; left: 20px;';
      case 'bottom-right':
        return 'bottom: 20px; right: 20px;';
      case 'bottom-left':
        return 'bottom: 20px; left: 20px;';
      default:
        return 'top: 20px; right: 20px;';
    }
  }
}

// Initialize
new EgoShieldContentScript();

Categories organized by primary harm mechanism.

from typing import List, Dict
from enum import Enum

# egoshield/security/credentials.py

"""
Secure storage of email credentials using system keyring.

Uses keyring library which interfaces with:
- macOS: Keychain
- Windows: Credential Vault
- Linux: Secret Service API (GNOME Keyring, KWallet)
"""

import keyring
from cryptography.fernet import Fernet
from cryptography.hazmat.primitives import hashes
from cryptography.hazmat.primitives.kdf.pbkdf2 import PBKDF2
import base64
import os
from typing import Optional

class CredentialManager:
    """
    Manages encrypted storage of sensitive credentials.
    
    Strategy:
    1. Use system keyring as primary storage (OS-managed encryption)
    2. Fallback: encrypted local storage with machine-specific key
    3. Never log credentials or include in error messages
    """
    
    SERVICE_NAME = "EgoShield"
    
    def __init__(self):
        self.keyring_available = self._check_keyring()
    
    def _check_keyring(self) -> bool:
        """Check if system keyring is available"""
        try:
            # Test keyring access
            keyring.get_keyring()
            return True
        except Exception:
            return False
    
    def store_email_credentials(self, username: str, password: str) -> bool:
        """
        Securely store email credentials.
        
        Returns True if successful, False otherwise.
        """
        try:
            if self.keyring_available:
                # Use system keyring (best security)
                keyring.set_password(self.SERVICE_NAME, f"email_{username}", password)
                return True
            else:
                # Fallback: encrypted file with machine key
                return self._store_encrypted_fallback(username, password)
        except Exception as e:
            # Never log the actual credentials
            print(f"Failed to store credentials: {type(e).__name__}")
            return False
    
    def get_email_credentials(self, username: str) -> Optional[str]:
        """
        Retrieve email password for given username.
        
        Returns password or None if not found.
        """
        try:
            if self.keyring_available:
                password = keyring.get_password(self.SERVICE_NAME, f"email_{username}")
                return password
            else:
                return self._get_encrypted_fallback(username)
        except Exception as e:
            print(f"Failed to retrieve credentials: {type(e).__name__}")
            return None
    
    def delete_email_credentials(self, username: str) -> bool:
        """Delete stored credentials"""
        try:
            if self.keyring_available:
                keyring.delete_password(self.SERVICE_NAME, f"email_{username}")
            else:
                self._delete_encrypted_fallback(username)
            return True
        except Exception:
            return False
    
    def _get_machine_key(self) -> bytes:
        """
        Generate encryption key derived from machine-specific data.
        
        Not perfect security, but better than plaintext.
        """
        # Use machine ID as salt
        import platform
        machine_id = f"{platform.node()}{platform.machine()}".encode()
        
        # Derive key using PBKDF2
        kdf = PBKDF2(
            algorithm=hashes.SHA256(),
            length=32,
            salt=machine_id,
            iterations=100000,
        )
        key = base64.urlsafe_b64encode(kdf.derive(machine_id))
        return key
    
    def _store_encrypted_fallback(self, username: str, password: str) -> bool:
        """Fallback: store in encrypted file"""
        key = self._get_machine_key()
        f = Fernet(key)
        
        encrypted = f.encrypt(password.encode())
        
        # Store in config directory
        cred_path = os.path.expanduser("~/.egoshield/credentials")
        os.makedirs(cred_path, mode=0o700, exist_ok=True)
        
        file_path = os.path.join(cred_path, f"{username}.enc")
        with open(file_path, 'wb') as file:
            file.write(encrypted)
        
        # Set restrictive permissions
        os.chmod(file_path, 0o600)
        return True
    
    def _get_encrypted_fallback(self, username: str) -> Optional[str]:
        """Retrieve from encrypted file"""
        file_path = os.path.expanduser(f"~/.egoshield/credentials/{username}.enc")
        
        if not os.path.exists(file_path):
            return None
        
        with open(file_path, 'rb') as file:
            encrypted = file.read()
        
        key = self._get_machine_key()
        f = Fernet(key)
        
        try:
            decrypted = f.decrypt(encrypted)
            return decrypted.decode()
        except Exception:
            return None
    
    def _delete_encrypted_fallback(self, username: str):
        """Delete encrypted credential file"""
        file_path = os.path.expanduser(f"~/.egoshield/credentials/{username}.enc")
        if os.path.exists(file_path):
            os.remove(file_path)

"""
Enforceable privacy guarantees.

These are hard constraints, not just best practices.
"""

from typing import Any, Dict
import logging

logger = logging.getLogger(__name__)

class PrivacyEnforcer:
    """
    Enforces privacy constraints at runtime.
    
    Design: Fail closed (deny operation if privacy check fails).
    """
    
    def __init__(self, config):
        self.config = config
        self.violations_logged = 0
    
    def check_network_request(self, url: str, data: Any) -> bool:
        """
        Verify network request complies with privacy policy.
        
        Rules:
        1. No requests to external domains (unless explicitly allowed)
        2. No user content sent to external services
        3. LLM requests must be to localhost
        
        Returns False if request should be blocked.
        """
        from urllib.parse import urlparse
        
        parsed = urlparse(url)
        
        # Rule 1: Only allow localhost or explicitly allowed domains
        if parsed.hostname not in ['localhost', '127.0.0.1', '::1']:
            if parsed.hostname not in self.config.privacy.get('allowed_external_domains', []):
                self._log_violation(f"Blocked external request to {parsed.hostname}")
                return False
        
        # Rule 2: No user content in external requests
        if parsed.hostname not in ['localhost', '127.0.0.1', '::1']:
            if isinstance(data, dict) and any(
                key in data for key in ['content', 'text', 'html', 'email', 'url']
            ):
                self._log_violation(f"Blocked user content in external request to {parsed.hostname}")
                return False
        
        return True
    
    def sanitize_for_storage(self, data: Dict) -> Dict:
        """
        Sanitize data before storing in database.
        
        Applies privacy settings:
        - URL anonymization (hashing)
        - Content truncation
        - PII removal
        """
        sanitized = data.copy()
        
        # URL anonymization
        if self.config.privacy.get('anonymize_urls', False):
            if 'url' in sanitized:
                sanitized['url'] = self._hash_url(sanitized['url'])
        
        # Content truncation
        if not self.config.privacy.get('store_full_content', False):
            if 'html' in sanitized:
                sanitized['html'] = None  # Don't store
            if 'text' in sanitized:
                sanitized['text'] = sanitized['text'][:500]  # Preview only
        
        # Remove email content if configured
        if self.config.privacy.get('dont_store_email_content', False):
            if 'source_type' in sanitized and sanitized['source_type'] == 'email':
                sanitized['content_preview'] = '[Email content not stored per privacy settings]'
                sanitized['html'] = None
                sanitized['text'] = None
        
        return sanitized
    
    def _hash_url(self, url: str) -> str:
        """Hash URL for anonymization"""
        import hashlib
        return 'hashed:' + hashlib.sha256(url.encode()).hexdigest()[:16]
    
    def _log_violation(self, message: str):
        """Log privacy violation"""
        logger.warning(f"PRIVACY VIOLATION: {message}")
        self.violations_logged += 1
        
        # Optional: terminate if too many violations (fail-safe)
        if self.violations_logged > 10:
            logger.critical("Too many privacy violations, shutting down")
            raise RuntimeError("Privacy violation threshold exceeded")

"""
Enforce no-telemetry policy.

This module provides compile-time and runtime checks to ensure
no telemetry code is present.
"""

import ast
import inspect
from typing import List

class TelemetryScanner:
    """
    Scan codebase for potential telemetry calls.
    
    Flags:
    - Network requests to non-localhost
    - Analytics libraries
    - Tracking pixels
    - Error reporting services
    """
    
    BANNED_IMPORTS = [
        'google.analytics',
        'mixpanel',
        'segment',
        'sentry_sdk',  # Error reporting
        'bugsnag',
        'rollbar',
        'amplitude',
        'heap',
    ]
    
    SUSPICIOUS_PATTERNS = [
        'analytics',
        'tracking',
        'telemetry',
        'phone-home',
        'usage-stats',
    ]
    
    def scan_file(self, filepath: str) -> List[str]:
        """
        Scan Python file for telemetry indicators.
        
        Returns list of warnings.
        """
        warnings = []
        
        with open(filepath, 'r') as f:
            content = f.read()
        
        # Parse AST
        try:
            tree = ast.parse(content)
        except SyntaxError:
            return [f"Syntax error in {filepath}"]
        
        # Check imports
        for node in ast.walk(tree):
            if isinstance(node, ast.Import):
                for alias in node.names:
                    if any(banned in alias.name for banned in self.BANNED_IMPORTS):
                        warnings.append(f"{filepath}: Banned import {alias.name}")
            
            elif isinstance(node, ast.ImportFrom):
                if node.module and any(banned in node.module for banned in self.BANNED_IMPORTS):
                    warnings.append(f"{filepath}: Banned import from {node.module}")
            
            # Check for suspicious function/variable names
            elif isinstance(node, ast.FunctionDef):
                if any(pattern in node.name.lower() for pattern in self.SUSPICIOUS_PATTERNS):
                    warnings.append(f"{filepath}: Suspicious function name {node.name}")
            
            elif isinstance(node, ast.Name):
                if any(pattern in node.id.lower() for pattern in self.SUSPICIOUS_PATTERNS):
                    warnings.append(f"{filepath}: Suspicious variable name {node.id}")
        
        return warnings
    
    def enforce_no_telemetry(self):
        """
        Enforce at runtime that no telemetry is possible.
        
        Patches urllib/requests to block non-localhost.
        """
        import urllib.request
        import urllib.error
        
        original_urlopen = urllib.request.urlopen
        
        def safe_urlopen(url, *args, **kwargs):
            from urllib.parse import urlparse
            parsed = urlparse(url)
            
            if parsed.hostname not in ['localhost', '127.0.0.1', '::1', None]:
                raise urllib.error.URLError(
                    "EgoShield: External network requests are disabled (no-telemetry policy)"
                )
            
            return original_urlopen(url, *args, **kwargs)
        
        urllib.request.urlopen = safe_urlopen
        
        # Patch requests library if present
        try:
            import requests
            original_request = requests.request
            
            def safe_request(method, url, *args, **kwargs):
                from urllib.parse import urlparse
                parsed = urlparse(url)
                
                if parsed.hostname not in ['localhost', '127.0.0.1', '::1']:
                    raise requests.exceptions.ConnectionError(
                        "EgoShield: External network requests are disabled (no-telemetry policy)"
                    )
                
                return original_request(method, url, *args, **kwargs)
            
            requests.request = safe_request
        except ImportError:
            pass  # requests not installed

"""
SQLite database management with connection pooling and WAL mode.
"""

import sqlite3
import threading
from contextlib import contextmanager
from typing import Optional, List, Dict, Any
import logging

logger = logging.getLogger(__name__)

class Database:
    """
    Thread-safe SQLite database wrapper.
    
    Features:
    - WAL mode for better concurrency
    - Connection pooling (one per thread)
    - Automatic migrations
    - Integrity checks
    - Backup support
    """
    
    def __init__(self, db_path: str):
        self.db_path = db_path
        self._local = threading.local()
        self._init_database()
    
    def _init_database(self):
        """Initialize database with schema"""
        with self.get_connection() as conn:
            # Enable WAL mode
            conn.execute("PRAGMA journal_mode=WAL")
            
            # Performance settings
            conn.execute("PRAGMA synchronous=NORMAL")  # Faster than FULL, safe in WAL mode
            conn.execute("PRAGMA cache_size=-64000")   # 64MB cache
            conn.execute("PRAGMA temp_store=MEMORY")   # Temp tables in memory
            
            # Run migrations
            self._run_migrations(conn)
    
    def get_connection(self) -> sqlite3.Connection:
        """
        Get thread-local connection.
        
        Each thread gets its own connection for thread-safety.
        """
        if not hasattr(self._local, 'connection'):
            conn = sqlite3.connect(
                self.db_path,
                check_same_thread=False,
                isolation_level=None  # Autocommit mode (we'll manage transactions explicitly)
            )
            conn.row_factory = sqlite3.Row  # Dict-like access to rows
            self._local.connection = conn
        
        return self._local.connection
    
    @contextmanager
    def transaction(self):
        """
        Transaction context manager.
        
        Usage:
            with db.transaction():
                db.execute("INSERT ...")
                db.execute("UPDATE ...")
            # Automatically commits or rolls back
        """
        conn = self.get_connection()
        conn.execute("BEGIN")
        try:
            yield conn
            conn.execute("COMMIT")
        except Exception:
            conn.execute("ROLLBACK")
            raise
    
    def execute(self, query: str, params: tuple = ()) -> sqlite3.Cursor:
        """Execute query with parameters"""
        return self.get_connection().execute(query, params)
    
    def fetchone(self, query: str, params: tuple = ()) -> Optional[Dict]:
        """Execute query and return first result as dict"""
        cursor = self.execute(query, params)
        row = cursor.fetchone()
        return dict(row) if row else None
    
    def fetchall(self, query: str, params: tuple = ()) -> List[Dict]:
        """Execute query and return all results as list of dicts"""
        cursor = self.execute(query, params)
        return [dict(row) for row in cursor.fetchall()]
    
    def _run_migrations(self, conn: sqlite3.Connection):
        """Run database migrations"""
        # Create migrations tracking table
        conn.execute("""
            CREATE TABLE IF NOT EXISTS schema_migrations (
                version INTEGER PRIMARY KEY,
                applied_at DATETIME DEFAULT CURRENT_TIMESTAMP
            )
        """)
        
        # Get current version
        cursor = conn.execute("SELECT MAX(version) FROM schema_migrations")
        current_version = cursor.fetchone()[0] or 0
        
        # Apply migrations
        migrations = self._get_migrations()
        for version, migration_sql in migrations:
            if version > current_version:
                logger.info(f"Applying migration {version}")
                conn.executescript(migration_sql)
                conn.execute("INSERT INTO schema_migrations (version) VALUES (?)", (version,))
    
    def _get_migrations(self) -> List[tuple]:
        """
        Return list of (version, sql) tuples.
        
        In production, these would be loaded from files.
        """
        return [
            (1, """
                -- Initial schema from section 5.1
                CREATE TABLE IF NOT EXISTS analyses (...);
                CREATE TABLE IF NOT EXISTS tactics (...);
                CREATE TABLE IF NOT EXISTS domain_stats (...);
                CREATE TABLE IF NOT EXISTS daily_reports (...);
                CREATE TABLE IF NOT EXISTS user_rules (...);
                CREATE TABLE IF NOT EXISTS settings (...);
                -- Indexes
                CREATE INDEX idx_analyses_timestamp ON analyses(timestamp);
                CREATE INDEX idx_analyses_domain ON analyses(domain);
                -- ... (full schema from section 5.1)
            """),
            
            # Future migrations would go here
            # (2, "ALTER TABLE analyses ADD COLUMN new_field TEXT"),
        ]
    
    def integrity_check(self) -> bool:
        """Run SQLite integrity check"""
        try:
            result = self.fetchone("PRAGMA integrity_check")
            return result['integrity_check'] == 'ok'
        except Exception as e:
            logger.error(f"Integrity check failed: {e}")
            return False
    
    def vacuum(self):
        """Reclaim unused space and defragment"""
        logger.info("Running VACUUM (this may take a while)...")
        self.execute("VACUUM")
    
    def backup(self, backup_path: str):
        """Create database backup"""
        import shutil
        # In WAL mode, must also copy WAL file
        shutil.copy2(self.db_path, backup_path)
        wal_path = self.db_path + '-wal'
        if os.path.exists(wal_path):
            shutil.copy2(wal_path, backup_path + '-wal')
        logger.info(f"Database backed up to {backup_path}")
    
    def close(self):
        """Close all connections"""
        if hasattr(self._local, 'connection'):
            self._local.connection.close()
            delattr(self._local, 'connection')

14.2 Repository Pattern

Python

# egoshield/storage/repositories/analysis_repo.py

"""
Analysis repository: CRUD operations for analyses and tactics.
"""

from typing import List, Dict, Optional
from datetime import datetime, timedelta
import uuid
import json

class AnalysisRepository:
    """Repository for analysis data"""
    
    def __init__(self, db):
        self.db = db
    
    def create(
        self,
        url: str,
        domain: str,
        source_type: str,
        title: Optional[str],
        content_preview: Optional[str],
        score: float,
        severity: str,
        tactic_count: int,
        tactics: List[Dict],
        processing_time_ms: int,
        llm_explanation: Optional[str] = None,
        llm_used: bool = False
    ) -> str:
        """Create new analysis record"""
        analysis_id = str(uuid.uuid4())
        
        with self.db.transaction():
            # Insert analysis
            self.db.execute("""
                INSERT INTO analyses (
                    id, url, domain, source_type, title, content_preview,
                    overall_score, severity, tactic_count, processing_time_ms,
                    llm_explanation, llm_used
                ) VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
            """, (
                analysis_id, url, domain, source_type, title, content_preview,
                score, severity, tactic_count, processing_time_ms,
                llm_explanation, llm_used
            ))
            
            # Insert tactics
            for tactic in tactics:
                tactic_id = str(uuid.uuid4())
                self.db.execute("""
                    INSERT INTO tactics (
                        id, analysis_id, tactic_type, confidence, severity,
                        evidence, explanation, location
                    ) VALUES (?, ?, ?, ?, ?, ?, ?, ?)
                """, (
                    tactic_id, analysis_id, tactic['type'], tactic['confidence'],
                    tactic['severity'], tactic['evidence'], tactic['explanation'],
                    tactic['location']
                ))
        
        return analysis_id
    
    def get_by_id(self, analysis_id: str) -> Optional[Dict]:
        """Get analysis with all tactics"""
        analysis = self.db.fetchone(
            "SELECT * FROM analyses WHERE id = ?",
            (analysis_id,)
        )
        
        if not analysis:
            return None
        
        # Get tactics
        tactics = self.db.fetchall(
            "SELECT * FROM tactics WHERE analysis_id = ? ORDER BY confidence DESC",
            (analysis_id,)
        )
        
        result = dict(analysis)
        result['tactics'] = tactics
        return result
    
    def query(
        self,
        domain: Optional[str] = None,
        source_type: Optional[str] = None,
        severity: Optional[str] = None,
        from_date: Optional[datetime] = None,
        to_date: Optional[datetime] = None,
        limit: int = 100,
        offset: int = 0
    ) -> tuple[List[Dict], int]:
        """Query analyses with filters"""
        
        # Build WHERE clause
        conditions = []
        params = []
        
        if domain:
            conditions.append("domain LIKE ?")
            params.append(f"%{domain}%")
        
        if source_type:
            conditions.append("source_type = ?")
            params.append(source_type)
        
        if severity:
            conditions.append("severity = ?")
            params.append(severity)
        
        if from_date:
            conditions.append("timestamp >= ?")
            params.append(from_date.isoformat())
        
        if to_date:
            conditions.append("timestamp <= ?")
            params.append(to_date.isoformat())
        
        where_clause = " AND ".join(conditions) if conditions else "1=1"
        
        # Get total count
        count_query = f"SELECT COUNT(*) as count FROM analyses WHERE {where_clause}"
        total = self.db.fetchone(count_query, tuple(params))['count']
        
        # Get results
        query = f"""
            SELECT * FROM analyses
            WHERE {where_clause}
            ORDER BY timestamp DESC
            LIMIT ? OFFSET ?
        """
        params.extend([limit, offset])
        results = self.db.fetchall(query, tuple(params))
        
        return results, total
    
    def delete_older_than(self, days: int) -> int:
        """Delete analyses older than N days"""
        cutoff = datetime.now() - timedelta(days=days)
        
        # Get count before delete
        count = self.db.fetchone(
            "SELECT COUNT(*) as count FROM analyses WHERE timestamp < ?",
            (cutoff.isoformat(),)
        )['count']
        
        # Delete (tactics will cascade due to FK)
        self.db.execute(
            "DELETE FROM analyses WHERE timestamp < ?",
            (cutoff.isoformat(),)
        )
        
        return count
    
    def get_summary_stats(
        self,
        from_date: Optional[datetime] = None,
        to_date: Optional[datetime] = None
    ) -> Dict:
        """Get summary statistics"""
        where_clause = "1=1"
        params = []
        
        if from_date:
            where_clause += " AND timestamp >= ?"
            params.append(from_date.isoformat())
        
        if to_date:
            where_clause += " AND timestamp <= ?"
            params.append(to_date.isoformat())
        
        stats = self.db.fetchone(f"""
            SELECT
                COUNT(*) as total_analyses,
                AVG(overall_score) as avg_score,
                MAX(overall_score) as max_score,
                SUM(CASE WHEN severity = 'high' THEN 1 ELSE 0 END) as high_severity_count,
                SUM(tactic_count) as total_tactics
            FROM analyses
            WHERE {where_clause}
        """, tuple(params))
        
        return dict(stats)




  ## 15. Logging, Observability & Debugging

### 15.1 Purpose

EgoShield must be observable enough for local debugging and performance tuning, but **must not become a privacy leak**. Logs should help answer:

- Did the backend daemon start correctly?
- Which detectors ran?
- How long did each detector take?
- Did Ollama respond?
- Did the browser extension reach the backend?
- Did the database write succeed?
- Which fallback path was used?

Logs must **never** contain full page HTML, full email bodies, credentials, cookies, auth tokens, or private message content.

---

### 15.2 Logging Principles

| Principle | Requirement |
|---|---|
| **Structured** | Logs emitted as JSON lines for easy parsing. |
| **Local-only** | Logs written only to local disk/stdout. No telemetry. |
| **Privacy-safe** | Store URLs optionally; never store full content by default. |
| **Configurable** | `debug`, `info`, `warning`, `error`. |
| **Rotated** | Log files should not grow forever. |
| **Performance-aware** | Logging must not block page analysis path. |

---

### 15.3 Log Levels

| Level | Use |
|---|---|
| `DEBUG` | Detector internals, pattern matches, timing breakdowns, LLM prompt summaries. Disabled by default. |
| `INFO` | Startup, shutdown, analysis complete, database initialized, extension connected. |
| `WARNING` | Ollama unavailable, detector timed out, backend degraded, privacy-protective fallback used. |
| `ERROR` | Database failure, malformed request, uncaught detector exception, migration failure. |
| `CRITICAL` | Privacy violation threshold exceeded, database corruption, unrecoverable startup failure. |

---

```python
# egoshield/observability/logging.py

import logging
import logging.handlers
import json
import os
import sys
import traceback
from datetime import datetime
from pathlib import Path
from typing import Any, Dict, Optional


class JsonFormatter(logging.Formatter):
    """
    JSON log formatter.

    Produces one JSON object per line:
    {
      "ts": "2026-05-29T12:34:56.123Z",
      "level": "INFO",
      "logger": "egoshield.detection.engine",
      "message": "analysis_complete",
      "analysis_id": "...",
      "duration_ms": 124
    }
    """

    RESERVED_ATTRS = {
        "name", "msg", "args", "levelname", "levelno", "pathname", "filename",
        "module", "exc_info", "exc_text", "stack_info", "lineno", "funcName",
        "created", "msecs", "relativeCreated", "thread", "threadName",
        "processName", "process"
    }

    def format(self, record: logging.LogRecord) -> str:
        log: Dict[str, Any] = {
            "ts": datetime.utcfromtimestamp(record.created).isoformat() + "Z",
            "level": record.levelname,
            "logger": record.name,
            "message": record.getMessage(),
        }

        # Include custom fields passed via `extra={...}`
        for key, value in record.__dict__.items():
            if key not in self.RESERVED_ATTRS and not key.startswith("_"):
                log[key] = self._sanitize(value)

        if record.exc_info:
            log["exception"] = self.formatException(record.exc_info)

        return json.dumps(log, ensure_ascii=False)

    def _sanitize(self, value: Any) -> Any:
        """
        Best-effort privacy sanitizer.

        Never log:
        - raw HTML
        - cookies
        - authorization headers
        - passwords
        - full email bodies
        """
        if isinstance(value, str):
            lowered = value.lower()

            if "password" in lowered:
                return "[REDACTED_PASSWORD]"

            if "authorization:" in lowered or "bearer " in lowered:
                return "[REDACTED_AUTH]"

            if "cookie:" in lowered:
                return "[REDACTED_COOKIE]"

            if len(value) > 500:
                return value[:497] + "..."

            return value

        if isinstance(value, dict):
            sanitized = {}
            for k, v in value.items():
                lk = str(k).lower()
                if lk in {"password", "token", "cookie", "authorization", "html", "full_content"}:
                    sanitized[k] = "[REDACTED]"
                else:
                    sanitized[k] = self._sanitize(v)
            return sanitized

        if isinstance(value, list):
            return [self._sanitize(v) for v in value[:20]]

        return value


def setup_logging(
    log_file: str,
    level: str = "info",
    max_bytes: int = 10 * 1024 * 1024,
    backup_count: int = 5,
) -> None:
    """
    Configure global structured logging.
    """

    log_path = Path(os.path.expanduser(log_file))
    log_path.parent.mkdir(parents=True, exist_ok=True)

    root = logging.getLogger()
    root.setLevel(parse_level(level))

    # Clear existing handlers
    root.handlers.clear()

    formatter = JsonFormatter()

    # File handler with rotation
    file_handler = logging.handlers.RotatingFileHandler(
        log_path,
        maxBytes=max_bytes,
        backupCount=backup_count,
        encoding="utf-8",
    )
    file_handler.setFormatter(formatter)
    file_handler.setLevel(parse_level(level))

    # Stdout handler
    stdout_handler = logging.StreamHandler(sys.stdout)
    stdout_handler.setFormatter(formatter)
    stdout_handler.setLevel(parse_level(level))

    root.addHandler(file_handler)
    root.addHandler(stdout_handler)

    logging.getLogger("uvicorn.access").setLevel(logging.WARNING)
    logging.getLogger("httpx").setLevel(logging.WARNING)


def parse_level(level: str) -> int:
    return {
        "debug": logging.DEBUG,
        "info": logging.INFO,
        "warning": logging.WARNING,
        "warn": logging.WARNING,
        "error": logging.ERROR,
        "critical": logging.CRITICAL,
    }.get(level.lower(), logging.INFO)
```

```python
# egoshield/detection/engine.py

import logging
import time

logger = logging.getLogger(__name__)


class DetectionEngine:
    def analyze(self, content, dom_signals, url):
        start = time.perf_counter()

        logger.debug(
            "analysis_started",
            extra={
                "url": self._safe_url(url),
                "content_chars": len(content.get("text", "")),
                "dom_signal_keys": list(dom_signals.keys()) if dom_signals else [],
            },
        )

        results = []

        for detector in self.detectors:
            detector_start = time.perf_counter()
            try:
                detector_results = detector.detect(content, dom_signals, url)
                elapsed_ms = int((time.perf_counter() - detector_start) * 1000)

                logger.debug(
                    "detector_completed",
                    extra={
                        "detector": detector.__class__.__name__,
                        "duration_ms": elapsed_ms,
                        "result_count": len(detector_results),
                    },
                )

                results.extend(detector_results)

            except Exception as exc:
                logger.error(
                    "detector_failed",
                    extra={
                        "detector": detector.__class__.__name__,
                        "error_type": type(exc).__name__,
                    },
                    exc_info=True,
                )
                continue

        total_ms = int((time.perf_counter() - start) * 1000)

        logger.info(
            "analysis_completed",
            extra={
                "duration_ms": total_ms,
                "tactic_count": len(results),
            },
        )

        return results

    def _safe_url(self, url: str) -> str:
        """
        Optionally strip query parameters to avoid logging tokens.
        """
        from urllib.parse import urlsplit, urlunsplit

        parts = urlsplit(url)
        return urlunsplit((parts.scheme, parts.netloc, parts.path, "", ""))
```

## 16. Testing Strategy

### 16.1 Testing Philosophy

EgoShield is a detection system. Its tests must validate both:

1. **Correctness** — obvious tactics are detected.
2. **Restraint** — benign content is not over-flagged.

False positives are as important as false negatives because a noisy cognitive firewall becomes unusable.

```python
# egoshield/backend/cli.py

import argparse
import webbrowser
from pathlib import Path

import uvicorn

from egoshield.config.config import Config
from egoshield.observability.logging import setup_logging


def main():
    parser = argparse.ArgumentParser(prog="egoshield")
    sub = parser.add_subparsers(dest="command")

    sub.add_parser("start", help="Start EgoShield backend daemon")
    sub.add_parser("init", help="Create default config and data directories")
    sub.add_parser("health", help="Check backend and Ollama health")
    sub.add_parser("reset-db", help="Delete and recreate local database")

    args = parser.parse_args()

    if args.command == "init":
        init_config()
    elif args.command == "health":
        health_check()
    elif args.command == "reset-db":
        reset_db()
    else:
        start()


def init_config():
    config_path = Config.default_config_path()
    cfg = Config.load(config_path)

    Path("~/.egoshield/data").expanduser().mkdir(parents=True, exist_ok=True)
    Path("~/.egoshield/logs").expanduser().mkdir(parents=True, exist_ok=True)

    print(f"✅ EgoShield initialized at {config_path}")


def start():
    cfg = Config.load()

    setup_logging(
        log_file=cfg.backend.log_file,
        level=cfg.backend.log_level,
    )

    print(f"🛡️ Starting EgoShield backend at http://{cfg.backend.host}:{cfg.backend.port}")

    if cfg.dashboard.auto_open_browser:
        webbrowser.open(f"http://{cfg.dashboard.host}:{cfg.dashboard.port}")

    uvicorn.run(
        "egoshield.backend.main:app",
        host=cfg.backend.host,
        port=cfg.backend.port,
        reload=False,
    )


def health_check():
    # Minimal CLI health check.
    import httpx

    try:
        response = httpx.get("http://127.0.0.1:9765/api/v1/health", timeout=2)
        print(response.json())
    except Exception as exc:
        print(f"❌ EgoShield backend unreachable: {exc}")


def reset_db():
    import os

    cfg = Config.load()
    db_path = Path(cfg.database.path).expanduser()

    if db_path.exists():
        db_path.unlink()

    print(f"✅ Deleted database: {db_path}")
    print("Run `egoshield start` to recreate schema.")


if __name__ == "__main__":
    main()
```

## 19. Performance Targets & Benchmarks

### 19.1 Performance Philosophy

EgoShield must not make browsing feel slow. The detection pipeline should be:

- Fast enough for real-time page analysis.
- Non-blocking for page rendering.
- Gracefully degraded when LLM is slow.
- Capable of skipping large/irrelevant pages.

The extension must collect data after page load or idle time, not block first paint.

---

### 19.2 Performance Targets

| Metric | Target | Notes |
|---|---:|---|
| Rule-based page analysis | `< 200ms P95` | Backend only, no LLM. |
| Rule-based page analysis | `< 100ms P50` | Typical page, <50k chars extracted. |
| Detector execution | `< 50ms per detector P95` | Regex/DOM signal detectors. |
| LLM arbiter latency | `< 30s P95` | Mistral 7B on M1-class machine. |
| DB insert per analysis | `< 5ms P95` | SQLite WAL mode. |
| DB query: recent analyses | `< 20ms P95` | With indexes. |
| Extension extraction overhead | `< 50ms P95` | Content script extraction only. |
| Overlay render time | `< 30ms P95` | Iframe UI. |
| Backend memory | `< 200MB idle` | Without spaCy large models. |
| Extension memory | `< 50MB` | Per tab overhead minimal. |
| Dashboard initial load | `< 1s` | Local static files. |

---

### 19.3 Latency Budget

```text
Target: < 200ms P95 rule-based page analysis

Browser content extraction:        0-50ms
HTTP request to localhost:         1-5ms
Backend request validation:        1-5ms
Normalization:                     5-20ms
Parallel detector execution:       20-100ms
Scoring:                           1-5ms
SQLite insert:                     1-5ms
HTTP response serialization:       1-5ms
Overlay render:                    10-30ms

Total typical:                     40-140ms
Target P95:                        <200ms
```

---

### 19.4 LLM Latency Budget

```text
Target: < 30s P95 LLM arbiter

Prompt construction:               <10ms
Content truncation/token estimate:  <20ms
Ollama request roundtrip:           1-30s
JSON parse/validation:              <10ms
Fallback if timeout:                30s max

LLM timeout default:                30s
```

LLM arbiter is never on the critical path for displaying the initial page. The extension may show:

1. Fast rule-based alert.
2. Later refined explanation if LLM completes.

### 19.5 Benchmark Harness

```python
# benchmarks/bench_detection.py

import time
import statistics
import json
from pathlib import Path

from egoshield.config.config import Config
from egoshield.detection.engine import DetectionEngine


def load_fixture(name: str):
    path = Path("tests/fixtures/pages") / name
    return path.read_text(encoding="utf-8")


def benchmark_detector(iterations: int = 100):
    cfg = Config.load()
    engine = DetectionEngine(cfg)

    html = load_fixture("ecommerce_urgency.html")

    content = {
        "headline": "Act now! Limited time offer ends tonight",
        "text": html,
        "html": html[:50000],
        "cta": "Buy Now",
        "footer": "",
    }

    dom_signals = {
        "pretickedBoxes": [],
        "countdowns": [],
        "confirmShaming": [],
    }

    durations = []

    for _ in range(iterations):
        start = time.perf_counter()
        results = engine.analyze(content, dom_signals, "https://example.com")
        elapsed_ms = (time.perf_counter() - start) * 1000
        durations.append(elapsed_ms)

    print({
        "iterations": iterations,
        "p50_ms": percentile(durations, 50),
        "p95_ms": percentile(durations, 95),
        "p99_ms": percentile(durations, 99),
        "mean_ms": statistics.mean(durations),
        "max_ms": max(durations),
        "result_count": len(results),
    })


def percentile(values, p):
    values = sorted(values)
    k = int(len(values) * p / 100)
    return values[min(k, len(values) - 1)]


if __name__ == "__main__":
    benchmark_detector()
```

## 20. Error Handling Strategy

### 20.1 Error Handling Principles

| Principle | Meaning |
|---|---|
| **Never break pages** | Extension failures must not affect website functionality. |
| **Fail open** | If EgoShield cannot analyze, browsing continues. |
| **Fail local** | Errors stay local and are not reported externally. |
| **Graceful degradation** | If LLM fails, use rule-based output. |
| **Detector isolation** | One detector failure does not stop others. |
| **User-visible only when useful** | Show errors only if user action is needed. |

### 20.3 Detector Failure Policy

```python
# egoshield/detection/engine.py

def run_detector_safely(detector, content, dom_signals, url):
    try:
        return detector.detect(content, dom_signals, url)
    except TimeoutError:
        logger.warning(
            "detector_timeout",
            extra={"detector": detector.__class__.__name__},
        )
        return []
    except Exception as exc:
        logger.error(
            "detector_exception",
            extra={
                "detector": detector.__class__.__name__,
                "error_type": type(exc).__name__,
            },
            exc_info=True,
        )
        return []
```

Rules:

- Detector exceptions are caught.
- The failed detector contributes zero results.
- Remaining detectors continue.
- User sees no error unless all detectors fail.

### 20.4 LLM Failure Policy

```python
# egoshield/arbiter/policy.py

class LLMFailurePolicy:
    """
    Defines how to behave when LLM is unavailable.
    """

    def handle_ollama_unreachable(self, tactics):
        return {
            "tactics": tactics,
            "llm_used": False,
            "fallback_level": "rule_based",
            "explanation": self.template_explanation(tactics),
            "warning": "Local LLM unavailable; showing rule-based analysis.",
        }

    def handle_timeout(self, tactics):
        return {
            "tactics": tactics,
            "llm_used": False,
            "fallback_level": "timeout",
            "explanation": self.template_explanation(tactics),
            "warning": "LLM explanation timed out; showing rule-based analysis.",
        }

    def handle_invalid_json(self, tactics):
        return {
            "tactics": tactics,
            "llm_used": False,
            "fallback_level": "invalid_llm_output",
            "explanation": self.template_explanation(tactics),
            "warning": "LLM returned invalid output; showing rule-based analysis.",
        }

    def template_explanation(self, tactics):
        if not tactics:
            return "No manipulation tactics detected."

        names = [t["type"].replace("_", " ") for t in tactics[:3]]
        return f"Detected tactics include: {', '.join(names)}."
```

### 20.5 Backend Unreachable Policy in Extension

```typescript
// extension/src/backend-client.ts

export class BackendClient {
  constructor(private baseUrl: string) {}

  async analyze(request: AnalysisRequest): Promise<AnalysisResult | null> {
    try {
      const controller = new AbortController();
      const timeout = setTimeout(() => controller.abort(), 3000);

      const response = await fetch(`${this.baseUrl}/api/v1/analyze`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify(request),
        signal: controller.signal,
      });

      clearTimeout(timeout);

      if (!response.ok) {
        console.warn('[EgoShield] Backend returned error', response.status);
        return null;
      }

      return await response.json();

    } catch (error) {
      console.warn('[EgoShield] Backend unreachable; page unaffected', error);
      return null;
    }
  }
}
```

Extension behavior:

- If backend is unreachable, do nothing by default.
- Badge may show `!` only if user enabled backend health warnings.
- Page functionality is never altered.

## 23. Open Questions & Future Work

### 23.1 Open Technical Questions

| Question | Status | Notes |
|---|---|---|
| How should EgoShield handle iframe content? | Open | Same-origin iframes are accessible; cross-origin iframes are restricted. Need per-frame injection strategy. |
| How should EgoShield handle Shadow DOM? | Open | Open shadow roots can be traversed; closed roots cannot. Need graceful partial analysis. |
| How to support mobile browsers? | Open | Firefox Android extension support may be possible; iOS Safari extension requires separate architecture. |
| How to support non-English languages? | Open | Regex detectors are English-first; multilingual support requires language-specific pattern packs. |
| How should user feedback improve detection? | Open | Could store local false-positive labels and tune thresholds, without cloud training. |
| Should detectors be community-extensible? | Open | Python plugins create supply-chain risk. Need sandbox or signed detector registry. |
| Can visual dark patterns be detected without screenshots? | Open | DOM heuristics catch many cases, but visual misdirection often needs layout/screenshot analysis. |
| Should EgoShield store full content for debugging? | Default no | Could be an explicit temporary debug mode with warnings. |
| How to handle pages with infinite scroll? | Open | Need incremental analysis or debounce strategy. |
| How to handle highly dynamic SPAs? | Open | MutationObserver can trigger rescans, but must avoid performance issues. |

### 23.6 Known v1.0 Limitations

1. **English-first detection**  
   Most linguistic patterns are English-only in v1.0.

2. **No screenshot analysis**  
   Visual dark patterns are approximated through DOM heuristics.

3. **Limited cross-origin iframe visibility**  
   Browser security prevents full inspection of some embedded content.

4. **LLM quality depends on local model**  
   Smaller models may produce weaker explanations.

5. **False positives are expected**  
   EgoShield must prioritize tunability and user feedback.

6. **Email monitoring is opt-in**  
   IMAP credentials are sensitive and not required for core functionality.

7. **Notification monitoring differs by OS**  
   Linux is easiest via D-Bus; Windows/macOS require platform-specific work.

8. **Not a fact-checker**  
   EgoShield detects persuasive tactics, not truthfulness.

9. **Not an ad blocker**  
   EgoShield annotates; it does not block or remove content.

10. **Not a legal compliance tool**  
   It can flag patterns, but cannot determine legal violations.
